Myspace API: App Data

These methods are relative to the tag 'App Data'

appdata.getAllApplicationData

Retrieves all application data for a specified user.

[object Object]

Authentication required

Parameter name Description Default Optionnal
personId The person's identifier. @me false
selector Indicates which set of individuals to query for activities. @friends and @all currently can be used interchangeably. false
appId The application's ID associated with the OAuth ConsumerKey/ConsumerSecret pair. @app false
fields The fields correspond to keys in the application data. One uses fields to only retrieve specific keys. If the request omits fields, all application data per user will be returned. true
