Myspace API: Search

These methods are relative to the tag 'Search'

search.getPeople

Returns search results for people.

[object Object]

Parameter name Description Default Optionnal
searchTerms Free form search terms or query words false
count Number of items to return. true
countryCode Country code to search with. Can be used instead of culture. true
culture The culture context of the search. Default is en-us. true
distance Distance away from location to return results. Default is in miles, but depending on culture specified, it can be in kilometers. true
gender The gender to filter on. Default is both. true
hasPhoto Filter for only those who have a photo. Default those with or without a photo. true
latitude Geographic latitude. Required to be combined with longitude and not combined with location field. true
location The location field, such as city, state and/or country. true
longitude Geographic longitude. Required to be combined with latitude and not combined with location field. true
minAge Minimum age for returned people. Same functionality as the MySpace website search. true
maxAge Maximum age for returned people. Same functionality as the MySpace website search. true
searchBy Which field the search should go through. Default is all of the fields. (yomi is only available to ja-jp culture). true
startPage Which page to start at for the results. true

search.getImages

Returns search results for images.

[object Object]

Parameter name Description Default Optionnal
searchTerms Free form search terms or query words false
count Number of items to return. true
culture The culture context of the search. Default is en-us. true
sortBy How to sort the images. Default is all. true
sortOrder Indicates whether to sort ascending or descending. Default is descending. true
startPage Which page to start at for the results. true

search.getVideos

Returns search results for videos.

[object Object]

Parameter name Description Default Optionnal
searchTerms Free form search terms or query words false
count Number of items to return. true
culture The culture context of the search. Default is en-us. true
startPage Which page to start at for the results. true
tag Determine if searching on tags, as opposed to contents. To enable tag search, set to 1. true
videoMode Search for specific video types. Two allowed values are videoMode=1 (music videos) or videoMode=2 (official). true
