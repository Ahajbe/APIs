Myspace API: Albums

These methods are relative to the tag 'Albums'

albums.getSupportedFields

Retrieves all supported fields.

Authentication required

Parameter name Description Default Optionnal
count Only returns the nearest multiple of 3 compared to the original value. true
fields The following field names are supported. Bold are default values.
caption, id, location, mediaItemCount, mediaType, ownerId, thumbnailUrl
@all (a special value that tells the REST endpoint to send back all fields for the album.)
 true
msPrivacyLevel MySpace specific field. May be one of the following values:
me - Album is private.
friendsonly - Album is visible to friends.
everyone - Album is visible to everyone.		 true
startIndex Indicates the index of the first item to retrieve from the query set. true

albums.getCurrentUserAlbums

@all (a special value that tells the REST endpoint to send back all fields for the album.)
 true
msPrivacyLevel MySpace specific field. May be one of the following values:
me - Album is private.
friendsonly - Album is visible to friends.
everyone - Album is visible to everyone.		 true
startIndex Indicates the index of the first item to retrieve from the query set. true
