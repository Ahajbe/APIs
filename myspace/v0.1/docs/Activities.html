Myspace API: Activities

These methods are relative to the tag 'Activities'

activities.getUserActivities

Retrieves all activities for the user or for the friends of the viewer.

Authentication required

Parameter name Description Default Optionnal
personId The person's identifier. @me false
selector Indicates which set of individuals to query for activities. @friends and @all currently can be used interchangeably. false
count Only returns the nearest multiple of 3 compared to the original value. true
fields The following field names are supported.
appid, body, externalid, id, mediaitems, postedtime, streamfaviconurl, title, titleid, userid,
@all (a special value that tells the REST endpoint to send back all fields for the activity.)
 true
startIndex Indicates the index of the first item to retrieve from the query set. true
updatedSince Indicates the date before which no activities should be returned. true
