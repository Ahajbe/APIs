Etsy_sandbox API: Shop

These methods are relative to the tag 'Shop'

getShop

Retrieves a Shop by id.

[object Object]

Authentication required

Parameter name Description Default Optionnal
shop_id false

findAllShops

Finds all Shops. If there is a keywords parameter, finds shops with shop_name starting with keywords.

[object Object]

Authentication required

Parameter name Description Default Optionnal
shop_name true
limit 25 true
offset 0 true

updateShop

Updates a Shop

[object Object]

Authentication required

Parameter name Description Default Optionnal
shop_id false
title true
announcement true
is_refusing_alchemy true
alchemy_message true
sale_message true
policy_welcome true
policy_payment true
policy_shipping true
policy_refunds true
policy_additional true

findAllUserShops

Retrieves a set of Shop objects associated to a User.

[object Object]

Authentication required

Parameter name Description Default Optionnal
user_id __SELF__ false
limit 25 true
offset 0 true
