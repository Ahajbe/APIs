Rhapsody API: Search

These methods are relative to the tag 'Search'

artist.search

Searches against the artist field in Rhapsody databases.

Parameter name Description Default Optionnal
query The text on which to search. false
size A numeral that sets the desired amount of returned results. true
pagination To get returned results starting at a particular offset, you need to pass in RhapsodyArtist.si=k%3Dnnn (or RhapsodyAlbum or RhapsodyTrack ) where nnn is the offset. For example: size=20&RhapsodyArtist.si=k%3D10
This example returns 20 records starting at number 10.		 true
