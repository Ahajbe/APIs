Wow API: Guilds

These methods are relative to the tag 'Guilds'

getGuild

Fetches a single guild at a time through an HTTP GET request to a url describing the guild profile resource. By default, a basic dataset will be returned and with each request and zero or more additional fields can be retrieved. The core dataset returned includes the guild's name, level, faction and achievement points.

Parameter name Description Default Optionnal
realm The realm name. false
guildName The guild name. false
fields Indicates that one or more of the optional datasets is to be retrieved. The list of the optional fields that can be requested:
- members: A list of characters that are a member of the guild.
- achievements: A set of data structures that describe the achievements earned by the guild.
