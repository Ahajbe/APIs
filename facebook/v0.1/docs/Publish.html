Facebook API: Publish

These methods are relative to the tag 'Publish'

postProfileIdFeed

Write to the given profile's feed/wall.

Authentication required

Parameter name Description Default Optionnal
profile_id me false
message The message I'm posting this from the Apigee API Console! false
picture If available, a link to the picture included with this post {attachment} true
link The link attached to this post http://app.apigee.com/console/facebook true
caption The caption of the link (appears beneath the link name) http://app.apigee.com/console/facebook true
name The name of the link Apigee's API Console true
description A description of the link (appears beneath the link caption) The API Console for Facebook allows you to quickly explore and test the Facebook API. true
