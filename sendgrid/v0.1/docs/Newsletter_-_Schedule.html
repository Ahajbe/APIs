Sendgrid API: Newsletter - Schedule

These methods are relative to the tag 'Newsletter - Schedule'

newsletter.schedule.add

Creates a new schedule event.

[object Object]

Authentication required

Parameter name Description Default Optionnal
name Newsletter to schedule delivery for (if Newsletter should be sent now, include no additional parameters). Must be an existing Newsletter. false
at Date/Time to schedule newsletter Delivery. Date/Time must be provided in ISO 8601 format (YYYY-MM-DD HH:MM:SS +-HH:MM). true
after Number of minutes until delivery should occur. Must be a positive integer. true

newsletter.schedule.get

Retrieves the scheduled event associated with a Newsletter.

[object Object]

Authentication required

Parameter name Description Default Optionnal
name Retrieve the delivery time scheduled for this Newsletter. Must be an existing Newsletter. false

newsletter.schedule.delete

Removes a scheduled event from a Newsletter.

[object Object]

Authentication required

Parameter name Description Default Optionnal
name Retrieve the delivery time scheduled for this Newsletter. Must be an existing Newsletter. false
