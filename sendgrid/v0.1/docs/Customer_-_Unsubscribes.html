user.unsubscribes.delete

Deletes unsubscribes. Since SendGrid does not deliver to unsubscribe addresses, users can remove unsubscribes from their list at any time if re-delivery to an unsubscribed address is desired.

[object Object]

Authentication required

Parameter name Description Default Optionnal
user The subuser we are retrieving unsubscribes from. Subuser must be registered under your account. false
task This will allow you to delete an unsubscribe record for the specified subuser. Must be set to delete delete false
email You must specify the unsubscribe record to remove. Must be an unsubscribe record. true
