Sendgrid API: Web - Invalid Emails

These methods are relative to the tag 'Web - Invalid Emails'

invalidemails.get

Retrieves a list of invalid emails with addresses and response codes, optionally with dates.

[object Object]

Authentication required

Parameter name Description Default Optionnal
date Retrieve the timestamp of the invalid email records. It will return a date in a MySQL timestamp format ��� YYYY-MM-DD HH:MM:SS. Must be set to 1. 1 true
days Number of days in the past for which to retrieve invalid emails (includes today). If specified, must be an integer greater than 0. true
start_date The start of the date range for which to retrieve invalid emails. Date must be in YYYY-MM-DD format and be earlier than the end_date parameter. true
end_date The end of the date range for which to retrieve invalid emails. Date must be in YYYY-MM-DD format and be later than the start_date parameter. true
limit Limit the number of results returned. true
offset Beginning point in the list to retrieve from. true
email Email addresses to search for. true

invalidemails.delete

Deletes an address from the Invalid Email list. Please note that if no parameters are submitted the ENTIRE list will be deleted.

[object Object]

Authentication required

Parameter name Description Default Optionnal
start_date The date to start deleting at. Date must be in YYYY-mm-dd format and be before the end_date parameter. true
end_date The date to end deleting at. Date must be in YYYY-mm-dd format and be after the start_date parameter. true
email Invalid Email address to remove. Must be a valid user account email. true
