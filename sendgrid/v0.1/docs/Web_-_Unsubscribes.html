Sendgrid API: Web - Unsubscribes

These methods are relative to the tag 'Web - Unsubscribes'

unsubscribes.get

Retrieves a list of Unsubscribes with addresses and optionally with dates.

Authentication required

Parameter name Description Default Optionnal
date Retrieve the timestamp of the unsubscribe records. It will return a date in a MySQL timestamp format ��� YYYY-MM-DD HH:MM:SS. Must be set to 1. true
days Number of days in the past for which to retrieve unsubscribes (includes today). If specified, must be an integer greater than 0. true
start_date The start of the date range for which to retrieve unsubscribes. Date must be in YYYY-MM-DD format and be earlier than the end_date parameter. true
end_date The end of the date range for which to retrieve unsubscribes. Date must be in YYYY-MM-DD format and be later than the start_date parameter. true
limit Limit the number of results returned. true
offset Beginning point in the list to retrieve from. true
email Email addresses to search for. true

unsubscribes.delete

Deletes an address from the Unsubscribe list. Please note that if no parameters are provided the ENTIRE list will be removed.

Authentication required

Parameter name Description Default Optionnal
start_date The date to start retrieving for. Date must be in YYYY-mm-dd format and be before the end_date parameter. true
end_date The date to end retrieving for. Date must be in YYYY-mm-dd format and be after the start_date parameter. true
email Unsubscribed email address to remove. Must be a valid user account email. true

unsubscribes.add

Adds an email address to the Unsubscribe list.

Authentication required

Parameter name Description Default Optionnal
email Email address to add to unsubscribe list. Must be a valid email address. false

unsubscribes.count

Counts the total number of unsubscribes.

Authentication required

Parameter name Description Default Optionnal
start_date The date to start counting from. Date must be in YYYY-mm-dd format and be before the end_date parameter. true
end_date The date to end counting from. Date must be in YYYY-mm-dd format and be after the start_date parameter. true
