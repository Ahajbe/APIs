Sendgrid API: Customer - Automatic Login

These methods are relative to the tag 'Customer - Automatic Login'

customer.geturl

Allows subusers to manage their account from your website using an iframe to our site. In order to login your subuser, you need to contact our web API to retrieve the unique URL to automatically login your subuser. Then display the generated URL to automatically login your subuser.

Authentication required

Parameter name Description Default Optionnal
user This is the subuser you will attempt to automatically login. Must be set in email format. false
password Authenticate the subuser with this API call. Your subuser password. true
