Sendgrid API: Newsletter - List Emails

These methods are relative to the tag 'Newsletter - List Emails'

newsletter.lists.email.add

Adds one or more emails to a Recipient List.

Authentication required

Parameter name Description Default Optionnal
list The list which you are adding email addresses too. Must be an existing Recipient List. false
data Specify the name, email address, and additional fields to add to the specified Recipient List. Must be a valid JSON string matching the required format (i.e. {"email":"address@domain.com","name":"contactName"}). Limited to a 1000 entries maximum. false

newsletter.lists.email.get

Gets the email addresses and associated fields for a Recipient List.

Authentication required

Parameter name Description Default Optionnal
list The list you are searching. Must be an existing Recipient List. false
email Email addresses to search for in the Recipient List. Must be an email address or list of addresses. true

newsletter.lists.email.delete

Removes one or more emails from a Recipient List.

Authentication required

Parameter name Description Default Optionnal
list The list which you are removing email addresses from. Must be an existing Recipient List. false
email Specify the email address or email addresses you wish to remove from the specified Recipient List. Must be an email or emails in the Recipient List. false
