Sendgrid API: Newsletter - Core

These methods are relative to the tag 'Newsletter - Core'

newsletter.add

Create a new Newsletter.

Authentication required

Parameter name Description Default Optionnal
identity The Identity that will be used for the Newsletter being created. false
name The name that will be used for the Newsletter being created. false
subject The subject that will be used for the Newsletter being created. false
text The text portion of the Newsletter being created. false
html The html portion of the Newsletter being created. false

newsletter.edit

Edit an existing Newsletter.

Authentication required

Parameter name Description Default Optionnal
name The name of the Newsletter being updated. false
newname The new name for the Newsletter being edited. false
identity The new Identity for the Newsletter being edited. false
subject The new subject that will be used for the Newsletter being edited. false
text The new text portion of the Newsletter being edited. false
html The new html portion of the Newsletter being edited. false

newsletter.get

Retrieves the contents of an existing newsletter.

Authentication required

Parameter name Description Default Optionnal
name The name of an existing Newsletter. false

newsletter.list

Retrieve the names of all existing Newsletters, or search for a specific newsletter.

Authentication required

Parameter name Description Default Optionnal
name Search to see if a specific Newsletter exists rather than a list of all Newsletters. false

newsletter.delete

Remove an existing Newsletter.

Authentication required

Parameter name Description Default Optionnal
name Remove the Newsletter with this name. false
