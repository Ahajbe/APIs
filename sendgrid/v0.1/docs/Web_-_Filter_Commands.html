Sendgrid API: Web - Filter Commands

These methods are relative to the tag 'Web - Filter Commands'

filter.getavailable

List all of the available apps. Note: The name entry is used in all the other API calls to identify a app. For a list of all the apps and parameters accesible with the Web API you can check the Filter Settings page, http://docs.sendgrid.com/documentation/api/web-api/filtersettings.

Authentication required

filter.activate

Activates an app.

Authentication required

Parameter name Description Default Optionnal
name Name of the app. false

filter.deactivate

Deactivates an app.

Authentication required

Parameter name Description Default Optionnal
name Name of the app. false

filter.getsettings

Get filter settings.

Authentication required

Parameter name Description Default Optionnal
name Name of the app. false
