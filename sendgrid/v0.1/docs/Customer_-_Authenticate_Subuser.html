Sendgrid API: Customer - Authenticate Subuser

These methods are relative to the tag 'Customer - Authenticate Subuser'

customer.auth

Authenticate a subuser on your website before displaying their account information so that you can have users manage their SendGrid account on your website completely.

Authentication required

Parameter name Description Default Optionnal
user Subuser that is registered under your account. false
password Password the subuser submitted. false
