Sendgrid API: Web - Bounces

These methods are relative to the tag 'Web - Bounces'

bounces.get

Retrieves a list of bounces with addresses and response codes, optionally with dates.

Authentication required

Parameter name Description Default Optionnal
date Retrieve the timestamp of the bounce records. It will return a date in a MySQL timestamp format ��� YYYY-MM-DD HH:MM:SS. Must be set to 1. true
days Number of days in the past for which to retrieve bounces (includes today). If specified, must be an integer greater than 0. true
start_date The start of the date range for which to retrieve bounces. Date must be in YYYY-MM-DD format and be earlier than the end_date parameter. true
end_date The end of the date range for which to retrieve bounces. Date must be in YYYY-MM-DD format and be later than the start_date parameter. true
limit Limit the number of results returned. true
offset Beginning point in the list to retrieve from. true
type The type of bounce to search for. Valid values: hard or soft. true
email Email addresses to search for. true

bounces.delete

Deletes an address from the Bounce list. Please note that if no parameters are specified the ENTIRE list will be deleted.

Authentication required

Parameter name Description Default Optionnal
start_date Date to start deleting from. Date must be in YYYY-mm-dd format and be before the end_date parameter. true
end_date Date to end deleting from. Date must be in YYYY-mm-dd format and be after the start_date parameter. true
type The type of bounce to be removed. Valid values: hard or soft. true
email Email bounce address to remove. true

bounces.count

Counts the total number of bounces.

Authentication required

Parameter name Description Default Optionnal
start_date Optional date to start counting from. Date must be in YYYY-mm-dd format and be before the end_date parameter. true
end_date date to end counting from. Date must be in YYYY-mm-dd format and be after the start_date parameter. true
type The type of bounce to search for. Valid values: hard or soft. true
