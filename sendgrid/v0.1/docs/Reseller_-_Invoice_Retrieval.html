Sendgrid API: Reseller - Invoice Retrieval

These methods are relative to the tag 'Reseller - Invoice Retrieval'

billing.list

Retrieve Invoice Numbers.

[object Object]

Authentication required

Parameter name Description Default Optionnal
start_date This is the start date to begin your invoice search from. Must be in YYYY-mm-dd format. true
end_date This is the end date for your invoice search. Must be in YYYY-mm-dd format. true
status This is used to specify what kind of invoice you would like returned. Must be either paid, unpaid, refunded, deferred, canceled. true
