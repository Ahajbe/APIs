Sendgrid API: Customer - Bounces

These methods are relative to the tag 'Customer - Bounces '

user.bounces.delete

Since SendGrid does not re-deliver to bounced addresses, users can remove bounces from their list at any time if redelivery to a bounced address is desired.

[object Object]

Authentication required

Parameter name Description Default Optionnal
user The subuser we are retrieving bounces from. Subuser must be registered under your account. false
task This will allow you to delete the bounces for the specified user. Must be set to delete. delete false
email Bounce email address you want to delete. true
