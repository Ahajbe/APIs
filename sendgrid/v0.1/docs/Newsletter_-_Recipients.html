Sendgrid API: Newsletter - Recipients

These methods are relative to the tag 'Newsletter - Recipients'

newsletter.recipients.add

Adds one or more Recipient List to a Newsletter.

Authentication required

Parameter name Description Default Optionnal
name This is the Newsletter to which you are adding Recipients Lists. Must be an existing Newsletter. false
list This is the Recipient List that will be added to the Newsletter. Must be one or more existing Recipient Lists. false

newsletter.recipients.get

Retrieves the Recipient Lists associated with a Newsletter.

Authentication required

Parameter name Description Default Optionnal
name Retrieve the Recipient Lists of an existing Newsletter. Must be an existing Newsletter. false

newsletter.recipients.delete

Removes a Recipient List from a Newsletter.

Authentication required

Parameter name Description Default Optionnal
name Newsletter to remove Recipient Lists from. Must be an existing Newsletter. false
list Recipient Lists to remove. Must be one or more existing Recipient Lists. false
