Sendgrid API: Web - Parse

These methods are relative to the tag 'Web - Parse'

parse.get

Pull down the settings already configured for parsing incoming email.

Authentication required

parse.set

Specify the hostname and url for parsing incoming emails.

Authentication required

Parameter name Description Default Optionnal
hostname The host name. false
url The URL. false
spam_check Whether to check or not for spam. true

parse.delete

Deletes the existing settings for parsing incoming emails.

Authentication required

Parameter name Description Default Optionnal
hostname The host name. false
