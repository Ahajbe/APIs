Sendgrid API: Newsletter - Identity

These methods are relative to the tag 'Newsletter - Identity'

newsletter.identity.add

Creates a new Identity.

[object Object]

Authentication required

Parameter name Description Default Optionnal
identity Create an Identity named this. false
name Specify the name to be used for this Identity. false
email Specify the email address to be used for this Identity. false
address Specify the physical address to be used for this Identity. false
city Specify the city to be used for this Identity. false
state Specify the state code to be used for this Identity. false
zip Specify the zip code to be used for this Identity. false
country Specify the country code to be used for this Identity. false

newsletter.identity.edit

Edits the contact information in an existing Identity.

[object Object]

Authentication required

Parameter name Description Default Optionnal
identity The Identity you wish to edit. Must be an existing Identity. false
newidentity Specify the new name to be used for this Identity. false
name Specify the new name to be used for this Identity. false
email Specify the new email address to be used for this Identity. false
address Specify the new physical address to be used for this Identity. false
city Specify the new physical address to be used for this Identity. false
state Specify the new physical address to be used for this Identity. false
zip Specify the new zip code to be used for this Identity. false
country Specify the new country code to be used for this Identity. false

newsletter.identity.get

Retrieves contact information associated with a particular Identity.

[object Object]

Authentication required

Parameter name Description Default Optionnal
identity The Identity you wish to retrieve. Must be an existing Identity. false

newsletter.identity.list

Lists all Identities on your account, or check if a particular Identity exists.

[object Object]

Authentication required

Parameter name Description Default Optionnal
identity Check for this particular Identity. (To list all Identities on your account exclude this parameter). true

newsletter.identity.delete

Removes an Identity from your account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
identity Remove the specified Identity from your account. Must be an existing Identity. false
