user.spamreports.delete

Deletes spam reports. Since SendGrid does not deliver to spam reported addresses, users can remove spam reports from their list at any time if re-delivery to a spam reported address is desired.

[object Object]

Authentication required

Parameter name Description Default Optionnal
user The subuser we are retrieving spam reports from. Subuser must be registered under your account. false
task This will allow you to delete a spam report for the specified user. Must be set to delete. delete false
email You must specify the spam reports to remove. Must be a spam reports. true
