Sendgrid API: Web - Blocks

These methods are relative to the tag 'Web - Blocks'

blocks.get

Retrieves a list of Blocks with addresses and response codes, optionally with dates.

Authentication required

Parameter name Description Default Optionnal
date Retrieve the timestamp of the Block records. It will return a date in a MySQL timestamp format ��� YYYY-MM-DD HH:MM:SS. Must be set to 1. true
days Number of days in the past for which to retrieve blocks (includes today). If specified, must be an integer greater than 0. true
start_date The start of the date range for which to retrieve blocks. Date must be in YYYY-MM-DD format and be earlier than the end_date parameter. true
end_date The end of the date range for which to retrieve blocks. Date must be in YYYY-MM-DD format and be later than the start_date parameter. true

blocks.delete

Deletes an address from the Block list.

Authentication required

Parameter name Description Default Optionnal
email Email block address to remove. Must be a valid user account email. false
