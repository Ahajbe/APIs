Skyrock API: Messages

These methods are relative to the tag 'Messages'

message_list_threads

Returns the authenticated user's message threads (15 threads per page).

[object Object]

Authentication required

Parameter name Description Default Optionnal
filter Thread selection filter (all/unread). all false
page Page number (optional). true

message_get_thread

Returns a message thread (15 messages per page).

[object Object]

Authentication required

Parameter name Description Default Optionnal
id_thread Message thread ID. false
page Page number (optional). true

message_set_thread_as_read

Marks a message thread as read.

[object Object]

Authentication required

message_delete_thread

Deletes a message thread.

[object Object]

Authentication required

message_delete_message

Deletes a message.

[object Object]

Authentication required

message_send_message

Sends a message to a single recipient.

[object Object]

Authentication required

message_create_thread

Creates a multi-recipient thread.

[object Object]

Authentication required

message_reply_thread

Replies to a thread.

[object Object]

Authentication required
