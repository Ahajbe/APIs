Skyrock API: Users

These methods are relative to the tag 'Users'

user_get

Returns an user's public data. If the call is authenticated and username is not provided, the method returns data about the current user's Skyrock account.

[object Object]

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true

user_list_friends

List of friends (paginated).

[object Object]

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true
page Page number. true
order Results order. Possible values:
- DEFAULT: default order.
- ALPHA: sort alphabetically.
- OLDEST_TO_NEWEST: sort by oldest to newest friends (by date of friendship).
- NEWEST_TO_OLDEST: sort by newest to oldest friends (by date of friendship).
- BEST_FIRST: default order, with best friends first.
- BEST_LAST: default order, with best friends last.
 DEFAULT true
