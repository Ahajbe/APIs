Skyrock API: Blogs

These methods are relative to the tag 'Blogs'

blog_get

Returns a blog public data. If the call is authenticated and username is not provided, the method returns data about the current user's Skyrock blog.

[object Object]

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true
output_format Defines which format to output, for specific rich text fields. Possible values: 'html' (default), 'bbcode'. html true

blog_set

Sets authenticated user's blog data. Each parameter is optional, you can either set one or many at once.

[object Object]

Authentication required

[object Object]

blog_list_posts

Returns an user's blog posts list (online only), paginated (10 articles per page).

[object Object]

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true
page Page number. 1 true
with_secret Allow authenticated blog owner to show secret posts (false by default). false true
output_format Defines which format to output, for specific rich text fields. Possible values: 'html' (default), 'bbcode'. html true
