Skyrock API: Moods

These methods are relative to the tag 'Moods'

moodListMoods

Returns an user's mood list.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true

moodGetMood

Returns an user last mood.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true

moodSetMood

Sets an user's mood.

Authentication required

moodDeleteMood

Deletes an user's mood.

Authentication required
