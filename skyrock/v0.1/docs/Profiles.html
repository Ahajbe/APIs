Skyrock API: Profiles

These methods are relative to the tag 'Profiles'

profileGet

Returns an user's profile public data.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true
output_format Defines which format to output, for specific rich text fields. Possible values: 'html' (default), 'bbcode'. html true

profileSet

Sets an user's profile public data.

Authentication required

profileAddAlbum

Adds an album to the profile.

Authentication required

profileListAlbums

Returns the list of albums associated with a profile. Main album has id_album = 1.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true

profileSetAlbum

Sets an album's title.

Authentication required

profileDeleteAlbum

Deletes an empty album. Main album (id_album = 1) cannot be deleted.

Authentication required

profileAddPicture

Adds a picture to the profile.

Authentication required

profileListPictures

Returns the list of pictures associated with a profile.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true
id_album Album ID. false

profileGetPicture

Returns a specified picture.

Authentication required

Parameter name Description Default Optionnal
id_album Album ID. false
id_picture Picture ID. false

profileSetPicture

Sets a picture's properties.

Authentication required

profileDeletePicture

Deletes a picture.

Authentication required

profileGetConfig

Returns settings about current user's profile.

Authentication required

profileSetConfig

Sets informations about a profile's (see get_property_values for possible values to set).

Authentication required

profileGetPropertyValues

Returns possible values for a given property. These properties can be set using set or set_config.

Authentication required

Parameter name Description Default Optionnal
property Property name. Possible values: gender, gender_preference, looks_for, status, town, country, department, occupation, place, personality, smoker, hair_color, eye_color, height, weight, block_friends, block_honors, block_groups, block_gifts, show_visits_counter, show_comments_counter, show_kiffs_counter, show_my_websites, show_mood, display_policy, allow_share, allow_facebook_like, allow_votes, allow_kiffs, default_page, show_me_page, show_news_page, show_photos_page, show_videos_page, allow_comments. false

profileSetTags

Sets an user profile's tags.

Authentication required

profileGetTags

Returns an user profile's tags.

Parameter name Description Default Optionnal
id_user User ID (optional if OAuthenticated). true
username Username (optional if OAuthenticated). true

profileGetTagsTypes

Returns user profile's tags types.
