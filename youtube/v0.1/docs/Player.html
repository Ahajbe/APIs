Youtube Player API

Youtube provide a Javascript API allowing you to integrate and manipulate a player inside your web application

Initialisation

To initialize a Youtube player, you have to give him the id of a Video on youtube : apis.youtube.player('g1BV7XTLUNg')

Some parameters can be given to chart()

Parameter Description Default Optionnal
idVideo The ID of the video you want to play g1BV7XTLUNg true
width The width of the player 100% false
height The height of the player 300 false

e.g : apis.google.player({idVideo: 'g1BV7XTLUNg', width: 250, height: 400})

Manipulation

Once your player instanciated and loaded: var player = apis.youtube.player('g1BV7XTLUNg');. You can manipulate it using the object stored in player variable

Youtube has the following interface :

Method Description Exemple
playVideo() Play the current video loaded 
player.playVideo();
pauseVideo() Pause the current video 
player.pauseVideo();
stopVideo() Stop the current video 
player.stopVideo();
seekTo(seconds, allowSeekAhead) Seek to 
pl.seekTo(42);
clearVideo() Clear the video loaded 
pl.clearVideo();
cue(url, startSeconds, suggestedQuality) Cue the url to be played after the current video. This url can be an or a videoId. The startSeconds is by default set to 0, and the suggestedQuality is auto set if none is given 
pl.cueVideo('g1BV7XTLUNg', 0, "medium");
load(url, startSeconds, suggestedQuality) Load another video in your player using is url or videoId 
player.load('g1BV7XTLUNg');
loadPlaylist(playlist, index, startSeconds, suggestedQuality) Load a playlist (using his id), and start to play at the index setted. If none is given, it will play the first video first 
var pl = apis.youtube.player();
pl.loadPlaylist({list: 'PL21087D879B1C8E93', index: 1, startSeconds: 1, suggestedQuality: "medium"});
cuePlaylist(playlist, index, startSeconds, suggestedQuality) Cue a playlist (using his id), and start to play at the index setted. If none is given, it will play the first video first 
var pl = apis.youtube.player();
pl.cuePlaylist({list: 'PL21087D879B1C8E93', index: 1, startSeconds: 10, suggestedQuality: "medium"});
nextVideo() When some videos are loaded, you can go to the next video using this method 
player.nextVideo();
previousVideo() When some videos are loaded, you can go to the previous video using this method 
pl.previousVideo();
playVideoAt(index) play the index video in the current playlist loaded 
player.playVideoAt(5);
mute() set mute 
player.mute();
unmute() set unmute 
player.unmute();
setVolume(vol) set the volume to the player. vol is an integer and have to be between 1 and 99 
pl.setVolume(75);
setLoop(activate) set loop playing to true or false. 
player.setLoop(true);
setShuffle(shufflePlaylist) set shuffle to true or false to play randomly a song in your playlist 
player.shuffle(true);
setPlaybackQuality(quality) set the quality of the video played 
var player = apis.youtube.player('g1BV7XTLUNg');
pl.setPlaybackQuality('')
