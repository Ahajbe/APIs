Blogger API: Comments Resource

These methods are relative to the tag 'Comments Resource'

blogger.blogs.comments.list

Retrieves the list of Comments for a Post.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog to fetch comments from. false
postId ID of the post to fetch posts from. false
fetchBodies Whether the body content of the comments is included. true
maxResults Maximum number of comments to include in the result. true
pageToken Continuation token if request is paged. true
startDate Earliest date of comment to fetch. true
fields Selector specifying which fields to include in a partial response. true

blogger.blogs.comments.get

Retrieves one Comments Resource by its commentId.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog to fetch comments from. false
postId ID of the post to fetch posts from. false
commentId The ID of the comment to get. false
fields Selector specifying which fields to include in a partial response. true
