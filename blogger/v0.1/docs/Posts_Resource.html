Blogger API: Posts Resource

These methods are relative to the tag 'Posts Resource'

blogger.blogs.posts.list

Retrieves a list of posts.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog to fetch posts from. false
fetchBodies Whether the body content of posts is included. true
maxResults Maximum number of posts to fetch. true
pageToken Continuation token if the request is paged. true
startDate Earliest post date to fetch. true
fields Selector specifying which fields to include in a partial response. true

blogger.blogs.posts.get

Retrieves one post by postId.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog to fetch the post from. false
postId The ID of the post. false
fields Selector specifying which fields to include in a partial response. true
