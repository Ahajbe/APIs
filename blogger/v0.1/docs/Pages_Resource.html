Blogger API: Pages Resource

These methods are relative to the tag 'Pages Resource'

blogger.blogs.pages.list

Retrieves the list of Pages for a Blog.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog to fetch pages from. false
fetchBodies Whether to retrieve the Page bodies. true
fields Selector specifying which fields to include in a partial response. true

blogger.blogs.pages.get

Retrieves one Pages Resource by its pageId.

Authentication required

Parameter name Description Default Optionnal
blogId ID of the blog containing the page. false
pageId The ID of the page to get. false
fields Selector specifying which fields to include in a partial response. true
