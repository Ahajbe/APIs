Blogger API: Blogs Resource

These methods are relative to the tag 'Blogs Resource'

blogger.users.blogs.list

Retrieves the list of Blogs for a User, given a userId.

[object Object]

Authentication required

Parameter name Description Default Optionnal
userId ID of the user whose blogs are to be fetched. false
fields Selector specifying which fields to include in a partial response. true

blogger.blogs.get

Retrieves a Blogs Resource's metadata by its blogId.

[object Object]

Authentication required

Parameter name Description Default Optionnal
blogId The ID of the blog to get. false
fields Selector specifying which fields to include in a partial response. true
