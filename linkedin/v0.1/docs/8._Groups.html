Linkedin API: 8. Groups

These methods are relative to the tag '8. Groups'

join-group

Join a group - this is the test group for people using the Groups API. Joining it will make the other calls work.

Authentication required

leave-group

Leave the API Groups Test Group.

Authentication required

get-groups

Get groups the member belongs to.

Authentication required

get-suggested-groups

Get suggested groups for the member.

Authentication required

group-settings

Get groups the member belongs to.

Authentication required

group-posts

Get posts for this group.

2 Legged Authentication works for this call when looking at open groups.

create-group-post

Create a post in a group.

Authentication required

like-group-post

Like a post in a group (this is the first post in the Groups API Test Group.)

Authentication required

comment-on-group-post

Comment on a post in a group (this is the first post in the Groups API Test Group.)

Authentication required
