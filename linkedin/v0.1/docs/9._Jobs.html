Linkedin API: 9. Jobs

These methods are relative to the tag '9. Jobs'

job-search

Search for jobs - by default it searches for jobs near the logged in member. Other parameters can be found on the job search documentation page on the developer portal.

Authentication required

recommended-jobs

Get recommended jobs for this member.

Authentication required

job-details

Retrieve information about a specific job. You need to input the ID for the job in the URL for this to work. Use the job search or recommended jobs calls above to get a job ID to use for this call.

Authentication required
