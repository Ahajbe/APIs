Orkut API: Acl Resource

These methods are relative to the tag 'Acl Resource'

orkut.acl.delete

Excludes an element from the ACL of the activity. The value "me" is valid for the parameter userId and means the authenticated user.

Authentication required

Parameter name Description Default Optionnal
activityId ID of the activity to remove. false
userId ID of the user to be removed from the activity. false
