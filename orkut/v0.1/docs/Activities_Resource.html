Orkut API: Activities Resource

These methods are relative to the tag 'Activities Resource'

orkut.activities.list

Retrieves a list of activities.

[object Object]

Authentication required

Parameter name Description Default Optionnal
userId The ID of the user whose activities will be listed. Can be me to refer to the viewer (i.e. the authenticated user). false
collection The collection of activities to list.
1. "stream", which shows activities from the user's friends. Applications can only view the stream of the authenticated user.
2. "all", which shows activities created by the user that are visible to the authenticated user.
3. "scraps", which shows the user's scrapbook. Access controls apply, so the authenticated user will see public scraps, scraps to all friends if he or she is a friend of the target user, and private scraps if he or she is in the Access control list
for the scrap.
false
hl Specifies the interface language (host language) of your user interface. true
maxResults The maximum number of activities to include in the response. (integer, 1-100) true
pageToken A continuation token that allows pagination. true
fields Selector specifying which fields to include in a partial response. true

orkut.activities.delete

Deletes an existing activity, if the access controls allow it. Currently the authenticated user is only allowed to delete their own activities.

[object Object]

Authentication required

Parameter name Description Default Optionnal
activityId ID of the activity to remove. false
