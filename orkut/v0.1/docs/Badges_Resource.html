Orkut API: Badges Resource

These methods are relative to the tag 'Badges Resource'

orkut.badges.list

Retrieves the list of visible badges of a user.

[object Object]

Authentication required

Parameter name Description Default Optionnal
userId The id of the user whose badges will be listed. Can be me to refer to caller. false
fields Selector specifying which fields to include in a partial response. true

orkut.badges.get

Retrieves a badge from a user.

[object Object]

Authentication required

Parameter name Description Default Optionnal
userId The ID of the user whose badges will be listed. Can be me to refer to caller. false
badgeId The ID of the badge that will be retrieved. false
fields Selector specifying which fields to include in a partial response. true
