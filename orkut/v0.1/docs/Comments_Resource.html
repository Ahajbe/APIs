Orkut API: Comments Resource

These methods are relative to the tag 'Comments Resource'

orkut.comments.list

Retrieves a list of comments, possibly filtered.

[object Object]

Authentication required

Parameter name Description Default Optionnal
activityId The ID of the activity containing the comments. false
fields Selector specifying which fields to include in a partial response. true
hl Specifies the interface language (host language) of your user interface. true
maxResults The maximum number of activities to include in the response. (integer, 1+) true
orderBy Sort search results. true
pageToken A continuation token that allows pagination. true

orkut.comments.insert

Inserts a new comment to an activity.

[object Object]

Authentication required

Parameter name Description Default Optionnal
activityId The ID of the activity containing the comments. false
fields Selector specifying which fields to include in a partial response. true
Content-Type Specifies Content Type. application/json false

orkut.comments.get

Retrieves an existing comment.

[object Object]

Authentication required

Parameter name Description Default Optionnal
commentId ID of the comment to get. false
hl Specifies the interface language (host language) of your user interface. true
fields Selector specifying which fields to include in a partial response. true

orkut.comments.delete

Deletes an existing comment.

[object Object]

Authentication required

Parameter name Description Default Optionnal
commentId ID of the comment to get. false
