Freebase API: Freebase

These methods are relative to the tag 'Freebase'

search

Find entities by keyword search.

Authentication required

Parameter name Description Default Optionnal
query The text you want to match against freebase entities. undefined false
callback JS method name for JSONP callbacks. undefined true
domain A comma separated list of domain IDs. Search results must include these domains. undefined true
exact Matches only the name, and keys 'exactly'. No normalization of any kind is done at indexing and query time. The text is only broken up on space characters. false true
filter A Filter s-expression. undefined true
html_escape Whether or not to escape entities. true true
indent Whether to indent the json. false true
limit Return up to this number of results. (integer >= 1) 20 true
mql_output A MQL query thats extracts entity information. undefined true
prefixed Whether or not to match by name prefix (used for autosuggest). false true
start Allows paging through results. (integer >= 0) 0 true
type A comma seperated list of type IDs. Search results must include these types. undefined true

mqlread

Retrieve detailed structured data about entities or collections of entities using the Metaweb Query Language (MQL).

Authentication required

Parameter name Description Default Optionnal
query A valid MQL query in the json format. undefined false
as_of_time Run the query as it would've been run at the specified point in time. Must be a valid timestamp (e.g. "2007-01-09T22" or "2007-02"). undefined true
callback JS method name for JSONP callbacks. undefined true
cursor Provides a mechanism for "paging" out results (can be used in conjunction with "limit" directive in MQL). No value (for the first request) OR a valid cursor returned by a previous request. undefined true
html_escape Whether or not to escape entities. true true
indent How many spaces to indent the JSON. (integer 0-10) false true
lang Return strings in the language requested (if a given object has no name in the requested language, null will be returned). /lang/en true
cost Whether or not to show the costs. false true
uniqueness_failure How MQL responds to uniqueness failures. hard true
