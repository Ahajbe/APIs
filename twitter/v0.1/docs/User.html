Twitter API: User

These methods are relative to the tag 'User'

showUser

Returns a variety of information about the user specified by the required user_id or screen_name parameter. The author's most recent Tweet will be returned inline when possible.

GET users/lookup is used to retrieve a bulk collection of user objects.

Parameter name Description Default Optionnal
user_id The ID of the user for whom to return results for. Either an id or screen_name is required for this method. true
screen_name The screen name of the user for whom to return results for. Either a id or screen_name is required for this method. true
include_entities The entities node will be disincluded when set to false. true

lookupUsers

Returns fully-hydrated user objects for up to 100 users per request, as specified by comma-separated values passed to the user_id and/or screen_name parameters.

This method is especially useful when used in conjunction with collections of user IDs returned from GET friends/ids and GET followers/ids.

GET users/show is used to retrieve a single user object.

Parameter name Description Default Optionnal
screen_name A comma separated list of screen names, up to 100 are allowed in a single request. You are strongly encouraged to use a POST for larger (up to 100 screen names) requests. true
user_id A comma separated list of user IDs, up to 100 are allowed in a single request. You are strongly encouraged to use a POST for larger requests. true
include_entities The entities node that may appear within embedded statuses will be disincluded when set to false. true

searchUsers

Provides a simple, relevance-based search interface to public user accounts on Twitter. Try querying by topical interest, full name, company name, location, or other criteria. Exact match searches are not supported.

Only the first 1,000 matching results are available.

Parameter name Description Default Optionnal
q The search query to run against people search. false
page Specifies the page of results to retrieve. true
count The number of potential user results to retrieve per page. This value has a maximum of 20. true
include_entities The entities node will be disincluded from embedded tweet objects when set to false. true

usersSuggestions

Access to Twitter's suggested user list. This returns the list of suggested user categories.

Parameter name Description Default Optionnal
lang Restricts the suggested categories to the requested language. The language must be specified by the appropriate two letter ISO 639-1 representation. Currently supported languages are provided by the apis.twitter.languages() API request. Unsupported language codes will receive English (en) results. If you use lang in this request, ensure you also include it when requesting the GET users/suggestions/:slug list. true

usersSuggestionsCategory

Access the users in a given category of the Twitter suggested user list.

It is recommended that applications cache this data for no more than one hour.

Parameter name Description Default Optionnal
slug The short name of list or a category twitter false
lang Restricts the suggested categories to the requested language. The language must be specified by the appropriate two letter ISO 639-1 representation. Currently supported languages are provided by the apis.twitter.languages() API request. Unsupported language codes will receive English (en) results. If you use lang in this request, ensure you also include it when requesting the GET users/suggestions/:slug list. true
