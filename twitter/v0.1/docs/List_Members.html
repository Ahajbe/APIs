Twitter API: List Members

These methods are relative to the tag 'List Members'

listmembers

Returns the members of the specified list. Private list members will only be shown if the authenticated user owns the specified list.

Authentication required

Parameter name Description Default Optionnal
format json false
include_entities true true
slug team false
cursor -1 true

listmembersshow

Check if the specified user is a member of the specified list.

Authentication required

Parameter name Description Default Optionnal
format json false
include_entities true true
slug team false
cursor -1 true

listmemberscreate

Add a member to a list. The authenticated user must own the list to be able to add members to it.

Authentication required

Parameter name Description Default Optionnal
format json false
slug team false
cursor -1 true

listmemberscreateall

Adds multiple members to a list, by specifying a comma-separated list of member ids or screen names. The authenticated user must own the list to be able to add members to it.

Authentication required

Parameter name Description Default Optionnal
format json false
slug team false

listmembersdestroy

Removes the specified member from the list. The authenticated user must be the list's owner to remove members from the list.

Authentication required
