Twitter API: Trends

These methods are relative to the tag 'Trends'

trendscurrent2

Returns the current top 10 trending topics on Twitter. The response includes the time of the request, the name of each trending topic, and query used on Twitter Search results page for that topic.

Authentication required

Parameter name Description Default Optionnal
format json false
exclude hashtags true

trendscurrent1

Returns the top 20 trending topics for each hour in a given day.

Authentication required

Parameter name Description Default Optionnal
format json false
exclude hashtags true
date 2009-03-19 true

trendsweekly

Returns the top 30 trending topics for each day in a given week.

Authentication required

Parameter name Description Default Optionnal
format json false
exclude hashtags true
date 2009-03-19 true
