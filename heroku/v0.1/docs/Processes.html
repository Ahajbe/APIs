Heroku API: Processes

These methods are relative to the tag 'Processes'

ListProcesses

List processes for an app.

[object Object]

Authentication required

Parameter name Description Default Optionnal
app The app name. false
Accept Content type. application/json false

runOne-offProcess

Run a one-off process.

[object Object]

Authentication required

Parameter name Description Default Optionnal
app The app name. false
Accept Content type. application/json false
command The command to run. false
attach true: use rendezvous to access stdin/stdout - : stream process output to the application log true

restartProcesses

Restart processes of an app. Specify either ps or type to restart only the specified processes. Leave both blank to restart the entire app.

[object Object]

Authentication required

Parameter name Description Default Optionnal
app The app name. false
ps The name of a process to restart. true
type The type of process to restart. true
Accept Content type. application/json false

stopProcesses

Stop processes of an app. Specify either ps or type to stop the specified processes.

[object Object]

Authentication required

Parameter name Description Default Optionnal
app The app name. false
Accept Content type. application/json false

scaleProcesses

Scale processes of an app.

[object Object]

Authentication required

Parameter name Description Default Optionnal
app The app name. false
type The type of process to scale. false
qty The desired number of processes of this type. false
Accept Content type. application/json false
