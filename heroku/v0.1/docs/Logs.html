Heroku API: Logs

These methods are relative to the tag 'Logs'

getAppLogs

Get logs for an app.

Authentication required

Parameter name Description Default Optionnal
app The app name. false
logplex Use logplex to stream logs (always true). true false
num The number of lines to display. true
ps Only display logs from a given process. true
source Only display logs from a given source. true
tail 1: continually stream logs - : display only num logs true
Accept Content type. application/json false
