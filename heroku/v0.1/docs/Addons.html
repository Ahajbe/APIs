Heroku API: Addons

These methods are relative to the tag 'Addons'

listAddons

List all available addons.

Authentication required

Parameter name Description Default Optionnal
Accept Content type. application/json false

listAppAddons

List addons installed on an app.

Authentication required

Parameter name Description Default Optionnal
app The app name. false
Accept Content type. application/json false

installAppAddon

Install an addon to an app.

Authentication required

Parameter name Description Default Optionnal
app The app name. false
addon the addon name. false
Accept Content type. application/json false

upgradeAppAddon

Upgrade an addon to an app.

Authentication required

Parameter name Description Default Optionnal
app The app name. false
addon the addon name. false
Accept Content type. application/json false

uninstallAppAddon

Uninstall an addon from an app.

Authentication required

Parameter name Description Default Optionnal
app The app name. false
addon the addon name. false
Accept Content type. application/json false
