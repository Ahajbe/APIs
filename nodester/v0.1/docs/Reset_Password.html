Nodester API: Reset Password

These methods are relative to the tag 'Reset Password'

postResetPassword

Sends a e-mail with a token to reset your user password

Parameter name Description Default Optionnal
user Username of the account to reset the password for
 false

putResetPassword

Updates your password using the provided token

Parameter name Description Default Optionnal
token Password reset token
 false
password The new password for the account
 false
