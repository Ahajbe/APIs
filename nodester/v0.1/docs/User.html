Nodester API: User

These methods are relative to the tag 'User'

postUser

Creates user account (pass in user and password and email and id_rsa.pub string) Ensure that all + in the ssh key are substituted for their %2B counter parts, else your key will break.

Parameter name Description Default Optionnal
coupon The user's coupon code
 false
user The username to register
 false
password The user's password
 false
email The user's email address
 false
rsakey The user's RSA SSH key
 false

putUser

Updates user account password and/or SSH key

Parameter name Description Default Optionnal
password The user's password
 true
rsakey The user's RSA SSH key
 true

deleteUser

Deletes the authenticated user's account

Authentication required
