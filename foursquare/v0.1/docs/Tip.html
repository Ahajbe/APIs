Foursquare API: Tip

These methods are relative to the tag 'Tip'

getTip

Gives details about a tip, including which users (especially friends) have marked the tip to-do or done.

Authentication required

Parameter name Description Default Optionnal
TIP_ID ID of tip to retrieve. undefined false

addTip

Allows you to add a new tip at a venue.

Authentication required

Parameter name Description Default Optionnal
venueId The venue where you want to add this tip. undefined false
text The text of the tip. undefined false
url A URL related to this tip. undefined true

getTips

Returns a list of tips near the area secified.

Authentication required

Parameter name Description Default Optionnal
ll Latitude and longitude of the user's location. undefined false
limit Number of results to return, up to 500. undefined true
offset Used to page through results. undefined true
filter If set to friends, only show nearby tips from friends. undefined true
query Only find tips matching the given term, cannot be used in conjunction with friends filter. undefined true

markToDoTip

Allows you to mark a tip to-do.

Authentication required

Parameter name Description Default Optionnal
TIP_ID The tip you want to mark to-do. undefined false

markdoneTip

Allows the acting user to mark a tip done.

Authentication required

Parameter name Description Default Optionnal
TIP_ID The tip you want to mark done. undefined false

unmarkTip

Allows you to remove a tip from your to-do list or done list.

Authentication required

Parameter name Description Default Optionnal
TIP_ID The tip you want to unmark. undefined false
