Foursquare API: Checkin

These methods are relative to the tag 'Checkin'

getCheckinDetails

Get details of a checkin.

Authentication required

Parameter name Description Default Optionnal
CHECKIN_ID The ID of the checkin to retrieve additional information for. undefined false

checkin

Allows you to check in to a place.

Authentication required

Parameter name Description Default Optionnal
venueId The venue where the user is checking in. No venueid is needed if shouting or just providing a venue name. Find venue IDs by searching or from historical APIs. undefined false
venue If are not shouting, but you don't have a venue ID or would rather prefer a 'venueless' checkin, pass the venue name as a string using this parameter. It will become an 'orphan' (no address or venueid but with geolat, geolong). undefined true
shout A message about your check-in. The maximum length of this field is 140 characters. undefined true
broadcast How much to broadcast this check-in, ranging from private (off-the-grid) to public,faceboook,twitter. Can also be just public or public,facebook, for example. If no valid value is found, the default is public. Shouts cannot be private. undefined false
ll Latitude and longitude of the user's location. undefined true
llAcc Accuracy of the user's latitude and longitude, in meters. undefined true
alt Altitude of the user's location, in meters. undefined true
altAcc Vertical accuracy of the user's location, in meters. undefined true

getCheckins

Returns a list of recent checkins from friends.

Authentication required

Parameter name Description Default Optionnal
ll Latitude and longitude of the user's location, so response can include distance. undefined true
limit Number of results to return, up to 500. undefined true
offset Used to page through results. undefined true
afterTimestamp Seconds after which to look for checkins, e.g. for looking for new checkins since the last fetch. If more than limit results are new since then, this is ignored. undefined true

addCheckinComment

Comment on a checkin-in

Authentication required

Parameter name Description Default Optionnal
CHECKIN_ID The ID of the checkin to add a comment to. undefined false
text The text of the comment, up to 200 characters. undefined false

removeCheckinComment

Remove a comment from a checkin, if the acting user is the author or the owner of the checkin.

Authentication required

Parameter name Description Default Optionnal
CHECKIN_ID The ID of the checkin to remove a comment from. undefined false
commentId The id of the comment to remove. undefined false
