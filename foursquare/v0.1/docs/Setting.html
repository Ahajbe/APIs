Foursquare API: Setting

These methods are relative to the tag 'Setting'

getSetting

Returns a setting for the acting user. SETTING_ID can be any of the three: receivePings, sendToTwitter, sendToFacebook

Authentication required

Parameter name Description Default Optionnal
SETTING_ID The name of a setting. undefined false

getAllSettings

Returns the settings of the acting user.

Authentication required

Parameter name Description Default Optionnal

setSetting

Change a setting for the given user.

Authentication required

Parameter name Description Default Optionnal
SETTING_ID Name of setting to change, sendToTwitter, sendToFacebook, receivePings. undefined false
value 1 for true, and 0 for false. undefined false
