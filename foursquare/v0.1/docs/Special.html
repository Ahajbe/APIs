Foursquare API: Special

These methods are relative to the tag 'Special'

getSpecialDetails

Gives details about a special, including text and unlock rules.

Authentication required

Parameter name Description Default Optionnal
SPECIAL_ID required ID of special to retrieve. undefined false

getSpecials

Returns a list of specials near the current location.

Authentication required

Parameter name Description Default Optionnal
ll Latitude and longitude to search near. undefined false
llAcc Accuracy of latitude and longitude, in meters. undefined true
alt Altitude of the user's location, in meters. undefined true
altAcc Accuracy of the user's altitude, in meters. undefined true
limit Number of results to return, up to 50. undefined true
