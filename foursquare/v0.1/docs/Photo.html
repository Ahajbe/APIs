Foursquare API: Photo

These methods are relative to the tag 'Photo'

getPhotoDetails

Get details of a photo.

Authentication required

Parameter name Description Default Optionnal
PHOTO_ID The ID of the photo to retrieve additional information for. undefined false

addPhoto

Allows users to add a new photo to a checkin, tip, or a venue in general.

Authentication required

Parameter name Description Default Optionnal
checkinId the ID of a checkin owned by the user. undefined true
tipId the ID of a tip owned by the user. undefined true
venueId the ID of a venue, provided only when adding a public photo of the venue in general, rather than a private checkin or tip photo using the parameters above. undefined true
broadcast Whether to broadcast this photo, ranging from twitter if you want to send to twitter, facebook if you want to send to facebook, or twitter,facebook if you want to send to both. undefined true
ll Latitude and longitude of the user's location. undefined true
llAcc Accuracy of the user's latitude and longitude, in meters. undefined true
alt Altitude of the user's location, in meters. undefined true
altAcc Vertical accuracy of the user's location, in meters. undefined true
