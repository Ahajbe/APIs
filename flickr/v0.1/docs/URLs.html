Flickr API: URLs

These methods are relative to the tag 'URLs'

flickr_urls_getGroup

Returns the url to a group's page.

[object Object]

Parameter name Description Default Optionnal
group_id The NSID of the group to fetch the url for.
 false

flickr_urls_getUserPhotos

Returns the url to a user's photos.

[object Object]

Parameter name Description Default Optionnal
user_id The NSID of the user to fetch the url for. If omitted, the calling user is assumed.
 true

flickr_urls_getUserProfile

Returns the url to a user's profile.

[object Object]

Parameter name Description Default Optionnal
user_id The NSID of the user to fetch the url for. If omitted, the calling user is assumed.
 true

flickr_urls_lookupGallery

Returns gallery info, by url.

[object Object]

Parameter name Description Default Optionnal
url The gallery's URL.
 false

flickr_urls_lookupGroup

Returns a group NSID, given the url to a group's page or photo pool.

[object Object]

Parameter name Description Default Optionnal
url The url to the group's page or photo pool.
 false

flickr_urls_lookupUser

Returns a user NSID, given the url to a user's photos or profile.

[object Object]

Parameter name Description Default Optionnal
url The url to the user's profile or photos page.
 false
