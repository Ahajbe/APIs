Flickr API: Favorites

These methods are relative to the tag 'Favorites'

flickr_favorites_add

Adds a photo to a user's favorites list.

[object Object]

Authentication required

Parameter name Description Default Optionnal
photo_id The id of the photo to add to the user's favorites.
 false

flickr_favorites_getList

Returns a list of the user's favorite photos. Only photos which the calling user has permission to see are returned.

[object Object]

Authentication required

Parameter name Description Default Optionnal
min_fave_date Minimum date that a photo was favorited on. The date should be in the form of a unix timestamp.
 true
max_fave_date Maximum date that a photo was favorited on. The date should be in the form of a unix timestamp.
 true
extras A comma-delimited list of extra information to fetch for each returned record. Currently supported fields are: description, license, date_upload, date_taken, owner_name, icon_server, original_format, last_update, geo, tags, machine_tags, o_dims, views, media, path_alias, url_sq, url_t, url_s, url_m, url_z, url_l, url_o
 true
per_page Number of photos to return per page. If this argument is omitted, it defaults to 100. The maximum allowed value is 500.
 100 true
page The page of results to return. If this argument is omitted, it defaults to 1.
 1 true
