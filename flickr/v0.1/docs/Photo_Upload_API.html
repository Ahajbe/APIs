Flickr API: Photo Upload API

These methods are relative to the tag 'Photo Upload API'

upload

Uploads a photo. Uploading apps can call the flickr.people.getUploadStatus method in the regular API to obtain file and bandwidth limits for the user.

Authentication required

Parameter name Description Default Optionnal
title The title of the photo. Optional.
 true
description A description of the photo. May contain some limited HTML. Optional.
 true
tags A space-seperated list of tags to apply to the photo. Optional.
 true
is_public Set to 0 for no, 1 for yes. Specifies who can view the photo. Optional.
 true
is_friend Set to 0 for no, 1 for yes. Specifies who can view the photo. Optional.
 true
is_family Set to 0 for no, 1 for yes. Specifies who can view the photo. Optional.
 true
safety_level Set to 1 for Safe, 2 for Moderate, or 3 for Restricted. Optional.
 true
content_type Set to 1 for Photo, 2 for Screenshot, or 3 for Other. Optional.
 true
hidden Set to 1 to keep the photo in global search results, 2 to hide from public searches. Optional.
 true
async Flickr can process photos in async mode, for applications that need to post multiple photos and don't want to wait around for each one to complete, leaving a socket connection open the whole time. Set to 1 for async mode, 0 for sync mode.
 true

replace

Replaces a photo that has already been uploaded to Flickr. Uploading apps can call the flickr.people.getUploadStatus method in the regular API to obtain file and bandwidth limits for the user.

Authentication required

Parameter name Description Default Optionnal
photo_id The ID of the photo to replace.
 false
async Photos may be replaced in async mode, for applications that don't want to wait around for an upload to complete, leaving a socket connection open the whole time. Processing photos asynchronously is recommended. Please consult the documentation for details. Set to 1 for async mode, 0 for sync mode.
 true
