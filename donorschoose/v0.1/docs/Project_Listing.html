Donorschoose API: Project Listing

These methods are relative to the tag 'Project Listing'

search

Project Search.

Authentication required

Parameter name Description Default Optionnal
callback If you wish to wrap your response in a callback function, include the name here. true
index By default, the start index of the result set is 0; for example, by setting index=10 and max=10, you can return rows 10-19. 0 true
max By default, the maximum number of projects returned is 10; you can request up to 50 rows at a time. 10 true
showSynopsis To show the synopsis for each project listing set this to true. true
showFacetCounts To show facet counts set this to true. They will appear after the project listings. true
showZeroFacetCountsForSchools Facet counts will be displayed for all schools--including those with zero live projects--if this parameter is set to true and showFacetCounts is set to true and school counts are displayed because there is a community parameter in the search parameters. true
keywords To search for a phrase, enclose it in quotes. It gets a list of projects where the keyword search matches any of the following fields:
- id
- title
