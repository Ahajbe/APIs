Shopping_com API: Category Tree

These methods are relative to the tag 'Category Tree'

categorytree

Returns information on a single category (e.g. name) or the relationship hierarchy between categories.

[object Object]

Authentication required

Parameter name Description Default Optionnal
categoryId The ID of the category for to return information for, as well as the information for all its immediate children (if any). To see all top level categories, use category ID '0', or simply omit the categoryId parameter altogether.
 true
showAllDescendants Specifies that information for all descendants of the specified category should be returned.
 true
