GETcategorytree

Returns information on a single category (e.g. name) or the relationship hierarchy between categories.

Test console
Description
Parameter
string

The ID of the category for to return information for, as well as the information for all its immediate children (if any). To see all top level categories, use category ID '0', or simply omit the categoryId parameter altogether.

categoryId

Optional

string

Specifies that information for all descendants of the specified category should be returned.
