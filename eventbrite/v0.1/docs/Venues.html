Eventbrite API: Venues

These methods are relative to the tag 'Venues'

venue_get

This method returns a single venue by id.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id The venue id.
 false

venue_new

This method creates a new venue. It returns the ID of the newly created venue.

[object Object]

Authentication required

Parameter name Description Default Optionnal
organizer_id The ID of the related organizer.
 false
venue The venue name.
 false
adress The venue address (line 1). Optional.
 true
adress_2 The venue address (line 2). Optional.
 true
city The venue city. Optional.
 true
region The venue state/province/county/territory depending on the country. 2-letter state code is required for US addresses.
 false
postal_code The postal code of the venue. Optional.
 true
country_code 2-letter country code, according to the ISO 3166 format.
 false
