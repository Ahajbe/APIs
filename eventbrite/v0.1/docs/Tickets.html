Eventbrite API: Tickets

These methods are relative to the tag 'Tickets'

ticketNew

This method creates new fixed-price or donation ticket types. It returns the ID of the newly created ticket.

Authentication required

Parameter name Description Default Optionnal
event_id The event ID. The event must have been previously created using /event_new.
 false
is_donation 0 for fixed-price tickets, 1 for donations. 0 will be used by default if not provided. Optional.
 true
name The ticket name.
 false
description The ticket description. Optional.
 true
price The ticket price. Enter 0.00 for free tickets. Leave blank for a donation.
 false
quantity The number of tickets available. Not required for donations.
 true

 true
include_fee 0 to add the Eventbrite service fee on top of ticket price, or 1 to include it in the ticket price. 0 will be used by default if not provided. Optional.
 true
min The minimum number of tickets per order. Optional.
 true
max The maximum number of tickets per order. Optional.
 true
