Eventbrite API: Users

These methods are relative to the tag 'Users'

userListVenues

This method lists the venues created by this user. Requires authentication.

Authentication required

userListTickets

This method lists the tickets purchased by the authenticated user. Each transaction is an order in our system and an order may contain one or more tickets. Tickets to free events are included.

Authentication required

userGet

This method returns the user account identified by the user_id or email request parameters. Visibility is limited to users and sub-users of the given user_key.

Authentication required

Parameter name Description Default Optionnal
user_id The ID of the subuser account. Optional.
 true
email The email address of the subuser account. Optional.
 true
