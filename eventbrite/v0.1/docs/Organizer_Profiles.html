organizer_new

Many event creators prefer having a specific person identified as the point of contact for their event. This person is usually someone like the event emcee, or an on-site contact that can help with attendee check-ins or other issues during the event. This method creates a new organizer, and returns the organizer_id for the newly created resource.

[object Object]

Authentication required

Parameter name Description Default Optionnal
name The organizer name.
 false
description The organizer description. Optional.
 true

organizer_update

This method updates an existing organizer. Only the fields passed as arguments will be modified. It returns the ID of the updated organizer.
