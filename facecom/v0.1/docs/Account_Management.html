Facecom API: Account Management

These methods are relative to the tag 'Account Management'

account.limits

Returns current rate limits for the account represented by the passed API key and Secret.

[object Object]

Authentication required

Parameter name Description Default Optionnal
callback JavaScript method to wrap a json-formatted response (for JSONP support, ignored if format=xml). true

account.users

Returns current users registered in the account's private namespaces. Users in a private namespace get registered implicitly through tags.save calls.

[object Object]

Authentication required

Parameter name Description Default Optionnal
namespaces Comma separated list of one or more private namespaces. false
callback JavaScript method to wrap a json-formatted response (for JSONP support, ignored if format=xml). true
