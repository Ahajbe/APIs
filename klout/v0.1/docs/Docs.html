Klout API

Method Description Exemple
score(users) This method allows you to retrieve a Klout score. users is a string containing comma-separated twitter usernames. 
apis.klout.score('webshell_,SteveMartinToGo')
showUser(users) This method allows you to retrieve user objects. users is a string containing comma-separated twitter usernames. 
apis.klout.showUser('webshell_,SteveMartinToGo')
topics(users) This method allows you to retrieve the top 3 topic objects for a given user. users is a string containing comma-separated twitter usernames. 
apis.klout.topics('webshell_,SteveMartinToGo')
influencedBy(users) This method allows you to retrieve up to 5 user score pairs for users that are influenced by the given influencer. users is a string containing comma-separated twitter usernames. 
apis.klout.influencedBy('webshell_,SteveMartinToGo')
influencerOf(users) This method allows you to retrieve up to 5 user score pairs for users that are influencers of the given user. users is a string containing comma-separated twitter usernames. 
apis.klout.influencerOf('webshell_,SteveMartinToGo')
