Github API: References

These methods are relative to the tag 'References'

getReference

Get a Reference

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
ref Must be formatted as `heads/branch`, not just `branch` undefined false

getSubReference

Get a Reference

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
ref Sub-namespace, e.g. `heads` or `tags` undefined false

updateReference

Update a Reference

Authentication required

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
ref Must be formatted as `heads/branch`, not just `branch` undefined false
sha SHA1 value to set this reference to undefined false
force Name of the fully qualified reference (ie: refs/heads/master). If it doesn’t start with ‘refs’ and have at least two slashes, it will be rejected undefined true

deleteReference

Delete a Reference

Authentication required

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
ref Must be formatted as `heads/branch`, not just `branch` undefined false

listReferences

Get all References. This will return an array of all the references on the system, including things like notes and stashes if they exist on the server. Anything in the namespace, not just heads and tags, though that would be the most common.

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
