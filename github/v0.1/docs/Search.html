Github API: Search

These methods are relative to the tag 'Search'

searchIssues

Find issues by state and keyword.

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
state `open` or `closed` undefined false
keyword Search term undefined false

searchRepos

Find repositories by keyword. Note, this legacy method does not follow the v3 pagination pattern. This method returns up to 100 results per page and pages can be fetched using the `start_page` parameter.

Parameter name Description Default Optionnal
keyword Search term undefined false
language Filter results by language undefined true
start_page Page number to fetch undefined true

searchUsers

Find users by keyword.

Parameter name Description Default Optionnal
keyword Search term undefined false
