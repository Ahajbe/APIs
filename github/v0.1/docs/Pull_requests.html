Github API: Pull requests

These methods are relative to the tag 'Pull requests'

listPullRequests

List pull requests

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
state `open` or `closed` to filter by state. open true

createPullRequest

Create a pull request. NOTE: `head` and `base` can be either a sha or a branch name. Typically you would namespace `head` with a user like this: `username:branch`.

Authentication required

Parameter name Description Default Optionnal
user Name of the user undefined false
repo Name of the repository undefined false
title Pull request title undefined false
body Pull request description undefined true
base The branch (or git ref) you want your changes pulled into. This should be an existing branch on the current repository. You cannot submit a pull request to one repo that requests a merge to a base of another repo. undefined false
head The branch (or git ref) where your changes are implemented. undefined false

base The branch (or git ref) you want your changes pulled into. This should be an existing branch on the current repository. You cannot submit a pull request to one repo that requests a merge to a base of another repo. undefined false
