Github API: Markdown

These methods are relative to the tag 'Markdown'

markdown

Render an arbritrary Markdown document

Authentication required

Parameter name Description Default Optionnal
text The Markdown text to render undefined false
mode The rendering mode. - `markdown` to render a document as plain Markdown, just like README files are rendered. - `gfm` to render a document as user-content, e.g. like user comments or issues are rendered. In GFM mode, hard line breaks are always taken into account, and issue and user mentions are linked accordingly. undefined true
context The repository context, only taken into account when rendering as `gfm` undefined true
