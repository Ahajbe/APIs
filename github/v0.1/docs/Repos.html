Github API: Repos

These methods are relative to the tag 'Repos'

listRepos

List repositories for the authenticated user.

Parameter name Description Default Optionnal
type `all`, `owner`, `public`, `private` or `member` all true
sort `created`, `updated`, `pushed` or `full_name` full_name true
direction `asc` or `desc`, default: when using `full_name`: `asc`, otherwise `desc`. undefined true

Parameter name Description Default Optionnal
name Repository name undefined false
description Repository description undefined true
homepage Repository homepage undefined true
private `true` to create a private repository, `false` to create a public one. Creating private repositories requires a paid GitHub account. false true
has_issues `true` to enable issues for this repository, `false` to disable them. true true
has_wiki `true` to enable the wiki for this repository, `false` to disable it. true true
has_downloads `true` to enable downloads for this repository, `false` to disable them. true true
team_id The id of the team that will be granted access to this repository. This is only valid when creating a repo in an organization. undefined true
