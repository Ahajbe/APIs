Delicious API: Tag Bundles

These methods are relative to the tag 'Tag Bundles'

tagBundlesAll

Retrieve all of a user's bundles.

[object Object]

Parameter name Description Default Optionnal
bundle Fetch just the named bundle. true

tagBundlesSet

Assign a set of tags to a single bundle, wipes away previous settings for bundle.

[object Object]

Parameter name Description Default Optionnal
bundle Name of the bundle. false
tags List of tags, space-separated. false

tagBundlesDelete

Delete a bundle.

[object Object]

Parameter name Description Default Optionnal
bundle Name of the bundle. false
