Delicious API: Posts

These methods are relative to the tag 'Posts'

postAdd

Add a post to Delicious.

[object Object]

Parameter name Description Default Optionnal
url The URL of the item. false
description The description of the item. false
extended Notes for the item. true
tags Tags for the item (space delimited). true
dt Datestamp of the item (format "CCYY-MM-DDThh:mm:ssZ"). Requires a LITERAL "T" and "Z" like in ISO8601 at http://www.cl.cam.ac.uk/~mgk25/iso-time.html for example: "1984-09-01T14:21:31Z" true
replace Don't replace post if given url has already been posted. no true
shared Make the item private. no true

postDelete

Delete a post from Delicious.

[object Object]

Parameter name Description Default Optionnal
url The URL of the item. false

postGet

Returns one or more posts on a single day matching the arguments. If no date or url is given, most recent date will be used.

[object Object]

Parameter name Description Default Optionnal
tag Filter by this tag (space delimited). true
dt Filter by this date, defaults to the most recent date on which bookmarks were saved. true
url Fetch a bookmark for this URL, regardless of date. Note: Be sure to URL-encode the argument value. true
hashes Fetch multiple bookmarks by one or more URL MD5s regardless of date, separated by URL-encoded spaces (ie. '+'). true
meta Include change detection signatures on each item in a 'meta' attribute. Clients wishing to maintain a synchronized local store of bookmarks should retain the value of this attribute - its value will change when any significant field of the bookmark changes. yes true

postRecent

Returns a list of the most recent posts, filtered by argument. Maximum 100.

[object Object]

Parameter name Description Default Optionnal
tag Filter by this tag. true
count Number of items to retrieve (Default:15, Maximum:100). 15 true

postDates

Returns a list of dates with the number of posts at each date.

[object Object]

Parameter name Description Default Optionnal
tag Filter by this tag. true

postAll

Returns all posts. Please use sparingly. Call the update function to see if you need to fetch this at all.

[object Object]

Parameter name Description Default Optionnal
tag Filter by this tag. true
start Start returning posts this many results into the set. true
results Return this many results. true
fromdt Filter for posts on this date or later. true
todt Filter for posts on this date or earlier. true
meta Include change detection signatures on each item in a 'meta' attribute. Clients wishing to maintain a synchronized local store of bookmarks should retain the value of this attribute - its value will change when any significant field of the bookmark changes. yes true

postAllHashes

Returns a change manifest of all posts. Call the update function to see if you need to fetch this at all.

[object Object]

postSuggest

Returns a list of popular tags, recommended tags and network tags for a user.

[object Object]

Parameter name Description Default Optionnal
url URL for which you'd like suggestions. false
