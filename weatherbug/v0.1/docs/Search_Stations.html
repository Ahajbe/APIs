Weatherbug API: Search Stations

These methods are relative to the tag 'Search Stations'

search.stations.zipcode

For a given zipcode, returns the weather stations in the area.

Authentication required

Parameter name Description Default Optionnal
zipCode 5 digits designation for U.S. cities only. false

search.stations.citycode

For a given citycode, returns the weather stations in the area.

Authentication required

Parameter name Description Default Optionnal
cityCode 5 digits designation for non-U.S. cities. false

search.stations.lat.long

For a given latitude/longitude, returns the weather stations in the area.

Authentication required

Parameter name Description Default Optionnal
lat Location of a place north or south of the equator. Minimum value: -90. Maximum value: 90. false
long Location of a place east or west of the prime meridian. Minimum value -180. Maximum value: 180. false
