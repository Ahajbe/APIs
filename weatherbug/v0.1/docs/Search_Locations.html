Weatherbug API: Search Locations

These methods are relative to the tag 'Search Locations'

search.location.zipcode

Returns the location based on U.S. Zip Code.

[object Object]

Authentication required

Parameter name Description Default Optionnal
SearchString Parameter used to search for locations. Only U.S. zipcodes and citynames can be searched. SearchString is not case-sensitive and accepts blank spaces. false

search.location.names

Returns the location based on U.S. and international city name. The soundex system allows searching for a city based on phonetics rather than spelling.
