Deezer API: Editorial

These methods are relative to the tag 'Editorial'

editorial

Return a list of editorials.

Parameter name Description Default Optionnal
index The index parameter specifies the index of the first matching result that should be included in the result set.
 true
nb-items the nb-items by page.
 true

getEditorial

Return the editorial defined in the id param.

Parameter name Description Default Optionnal
id The id of the editorial.
 false
index The index parameter specifies the index of the first matching result that should be included in the result set.
 true
nb-items the nb-items by page.
 true

getEditorialSelection

Return a list of albums selected every week by the Deezer Team.

Parameter name Description Default Optionnal
id The id of the editorial.
 false
index The index parameter specifies the index of the first matching result that should be included in the result set.
 true
nb-items the nb-items by page.
 true
