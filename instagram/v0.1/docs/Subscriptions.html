Instagram API: Subscriptions

These methods are relative to the tag 'Subscriptions'

post_subscriptions_user

Creates a subscription to receive notifications when users who have registered with your application post new photos.

[object Object]

Authentication required

Parameter name Description Default Optionnal
verify_token false
client_id Client ID of your application.
 false
client_secret Client secret of your application.
 false

delete_subscriptions

Deletes a subscription by object type or ID.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id The ID of the subscription you want to delete. Either this or "object" is required.
 true
object The object type of the subscription you want to delete, or "all" to clear all subscriptions.
 true
client_id Client ID of your application.
 false
client_secret Client secret of your application.
 false
