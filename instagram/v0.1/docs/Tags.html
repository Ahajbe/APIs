Instagram API: Tags

These methods are relative to the tag 'Tags'

getTags

Get information about a tag object.

Authentication required

Parameter name Description Default Optionnal
tag-name The name of the tag to reference.
 false

getTagsMediaRecent

Get a list of recently tagged media.

Authentication required

Parameter name Description Default Optionnal
tag-name The name of the tag to reference.
 false
max_id Return media after this max_id
 true
min_id Return media before this min_id
 true

getTagsSearch

Search for tags by name.

Authentication required

Parameter name Description Default Optionnal
q Valid tag name without a leading #. (eg. snow, nofilter)
 false
