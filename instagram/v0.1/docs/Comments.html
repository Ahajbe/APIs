Instagram API: Comments

These methods are relative to the tag 'Comments'

getMediaComments

Get a full list of comments on a media. Required scope: comments

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false

postMediaComments

Create a comment on a media. Required scope: comments

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false

deleteMediaComments

Remove a comment either on the authenticated user's media or authored by the authenticated user. Required scope: comments

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false
comment-id The ID of the comment to delete.
 false
