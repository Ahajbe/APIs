Instagram API: Relationships

These methods are relative to the tag 'Relationships'

getUsersFollows

Get the list of users this user follows. Required scope: relationships

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user to reference.
 false

getUsersFollowedBy

Get the list of users this user is followed by. Required scope: relationships

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user to reference.
 false

getIncomingRequests

List the users who have requested this user's permission to follow. Required scope: relationships

Authentication required

getRelationship

Get information about the current user's relationship (follow/following/etc) to another user. Required scope: relationships

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user to reference.
 false

postRelationship

Modify the relationship between the current user and the target user. Required scope: relationships

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user to reference.
 false
