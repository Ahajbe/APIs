Instagram API: Likes

These methods are relative to the tag 'Likes'

getMediaLikes

Get a list of users who have liked this media. Required scope: likes

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false

postMediaLikes

Set a like on this media by the currently authenticated user. Required scope: likes

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false

deleteMediaLikes

Remove a like on this media by the currently authenticated user. Required scope: likes

Authentication required

Parameter name Description Default Optionnal
media-id The ID of the media object to reference.
 false
