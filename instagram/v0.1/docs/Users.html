Instagram API: Users

These methods are relative to the tag 'Users'

getUsers

Get basic information about a user.

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user whose information to retrieve, or "self" to reference the currently logged-in user.
 false

getUsersFeed

See the authenticated user's feed.

Authentication required

Parameter name Description Default Optionnal
max_id Return media earlier than this max_id
 true
min_id Return media later than this min_id
 true
count Count of media to return
 true

getUsersMediaRecent

Get the most recent media published by a user.

Authentication required

Parameter name Description Default Optionnal
user-id The ID of the user whose recent media to retrieve.
 false
max_id Return media earlier than this max_id
 true
min_id Return media later than this min_id
 true
count Count of media to return
 true
min_timestamp Return media after this UNIX timestamp
 true
max_timestamp Return media before this UNIX timestamp
 true

getUsersLikedFeed

See the authenticated user's list of media they've liked.

Authentication required

Parameter name Description Default Optionnal
max_like_id Return media liked before this id
 true
count Count of media to return
 true

getUsersSearch

Search for a user by name.

Authentication required

Parameter name Description Default Optionnal
q A query string.
 false
count Number of users to return
 true
