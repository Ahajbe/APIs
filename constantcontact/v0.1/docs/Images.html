Constantcontact API: Images

These methods are relative to the tag 'Images'

images.list

Obtaining a list of images in a folder.

[object Object]

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
folder-id Represents an individual folder. false
Accept Specifies Accept. application/atom+xml false
Content-Type Specifies Content Type. application/atom+xml false
