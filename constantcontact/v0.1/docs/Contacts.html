Constantcontact API: Contacts

These methods are relative to the tag 'Contacts'

contacts.list

Retrieve all contacts for the account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false

contacts.add

Add a new contact to the account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
Content-Type Specifies Content Type. application/atom+xml false

contacts.search.by.email

Search for a specific contact by email address.

[object Object]

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
email Email address of a Contact. false

contacts.search.by.list.types

Search for contacts on specific list types that have been updated since a specified date.

[object Object]

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
updatedsince Specified date. The date has to be in UTC in the format specified by the Atom spec (i.e. 2008-07-23T14:21:06.407Z). false
listtype Specific list type. false
