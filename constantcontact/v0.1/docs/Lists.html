Constantcontact API: Lists

These methods are relative to the tag 'Lists'

lists.list

Retrieve the list of all mailing lists for the account.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false

lists.add

Create a new mailing list.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
Content-Type Specifies Content Type. application/atom+xml false

lists.get

Retrieving an Individual List.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
list-id The list identifier. false

lists.update

Updating a List.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
list-id The list identifier. false
Content-Type Specifies Content Type. application/atom+xml false

lists.delete

Deleting a List.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
list-id The list identifier. false

lists.members

Retrieving a contacts collection from a specific list.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
list-id The list identifier. false
