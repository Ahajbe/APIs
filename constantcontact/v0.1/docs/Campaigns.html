Constantcontact API: Campaigns

These methods are relative to the tag 'Campaigns'

campaigns.list

List all campaigns.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false

campaigns.search.status

Search for campaigns with a given status.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
status Campaign status. false
refresh To initiate re-calculation of campaign results, you must use refresh= true. true

campaigns.refresh.results

Manually refreshing campaign Results.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
refresh To initiate re-calculation of campaign results, you must use refresh= true. true false

campaigns.add

Create a new campaign.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
Content-Type Specifies Content Type. application/atom+xml false

campaigns.get

Return details for the specified campaign.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
campaign-id Represents an individual Campaign entry. false

campaigns.update

Update selected information for the specified campaign.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
campaign-id Represents an individual Campaign entry. false
Content-Type Specifies Content Type. application/atom+xml false

campaigns.delete

Removing a Campaign.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
campaign-id Represents an individual Campaign entry. false

campaigns.events

The campaign events resource that contains send, open, unsubscribe, click, bounce and forward events for a given campaign.

Authentication required

Parameter name Description Default Optionnal
username The account owner's Constant Contact user name. Value must be in lower case. false
campaign-id Represents an individual Campaign entry. false
