Google Maps API

Integrate a Google Maps with Webshell is pretty easy

Initialisation

To initialize a google maps : apis.google.maps()

Some parameters can be given to maps()

Parameter Description Default Optionnal
width The width of the maps Container width false
height The height of the maps 300 false

e.g : apis.google.maps({width: 550, height: 400})

Manipulation

Once your Google Maps instanciated : var m = apis.google.maps();. You can manipulate it using the object stored in m variable

Google Maps has the following interface :

Method Description Exemple
center(address) Center the google maps with the address as center 
var m = apis.google.maps();
m.center('paris');
zoom(level) Set the Zoom level for your Google Maps. zoom is a number between 1 and 18 
var m = apis.google.maps();
m.zoom(16);
addMarker(location) Add a marker on the Google Maps at a specified location. That location can be an address or a couple of latitude, longitude. This marker can have a title and an icon 
var m = apis.google.maps();
m.addMarker('Palais brongniart');
or
var m = apis.google.maps();
m.addMarker({
    address: "San Francisco, Fort Mason",
    icon: "http://www.perigorddecouverte.com/marker_fermier.png",
    title: "Would be nice to be there"
});
