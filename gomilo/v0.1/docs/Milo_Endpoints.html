Gomilo API: Milo Endpoints

These methods are relative to the tag 'Milo Endpoints'

store_addresses

The Store Addresses endpoint returns information about stores in a particular location. You must specify a location
(see Specifying Location in Requests), and can optionally limit the results to stores from a particular merchant.

products

The Products endpoint returns detailed product data. It accepts optional location parameters (see Specifying Location in Requests)
and has two methods of specifying which products to return (by ID or by search parameters).
You can also optionally specify a sort order and pagination options.

availability

The Availability endpoint provides specific availability (inventory) data on a per-store, per-product basis. It requires a location (see Location) and a product ID to be specified and can optionally take a JSONP callback function name. Doing so will cause the response to be in HTML format, consisting of script tags wrapping the callbacks. The default format is a sequence of JSON expressions with no wrapping. The radius parameter has no effect for this endpoint.
