Gravatar API

Gravatar provide a service to simplify the use of avatar on the cloud. Once user created an account on gravatar, each website who have his mail address can retrieve the avatar of the user

Display gravatar from an email

: apis.gravatar('team@webshell.io')

Gravatar have few options :

Parameter Description Default Optionnal
mail The mail of the user none true
size The size of the generated avatar false
default the url of the default avatar if the mail doesn't match any existing email address false
