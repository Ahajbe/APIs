Infoconnect API: Companies

These methods are relative to the tag 'Companies'

getCompany

Retrieves details of the Company identified by the resource.

Authentication required

Parameter name Description Default Optionnal
id A nine to ten digit Infogroup identification number. A number referencing the company assigned by Infogroup. This number will always refer to a single company and the reference will never change, unless the resource is permanently removed. Requesting a record by {id} will always return zero or one response. false
resourcetype Identifies the resource type for the response. The default value basic returns a base resource type that is valid across all methods; the resource type parameter is not needed to produce this response. To get a more detailed response set the resource type parameter to enhanced. To retrieve resources with Id's only, set the resource type parameter to core. basic true
Content-Type The content type. application/json false

getCompanies

Retrieves lists of companies.

Authentication required

Parameter name Description Default Optionnal
resourcetype Identifies the resource type for the response. The default value basic returns a base resource type that is valid across all methods; the resource type parameter is not needed to produce this response. To get a more detailed response set the resource type parameter to enhanced. To retrieve resources with Id's only, set the resource type parameter to core. basic true
city Filters through the Companies collection by city name. Omaha true
companyname Filters through the Companies collection and returns resources by company name. ABC true
ein Filters through the Companies collection and returns resources with an Employer Identification Number (EIN). 521492843 true
email Filters through the Companies collection and returns resources with a case-insensitive email address match at the level. johndoe@abcco.com true
firstname Filters through the Companies collection by the primary contact's first name. bill true
lastname Filters through the Companies collection by the primary contact's last name. doe true
phone Filters through the Companies collection and returns resources with an ten-digit phone number match. 5558675309 true
postalcode Filters through the Companies collection by ZIP code or Postal code. 68046 true
stateprovince Filters through the Companies collection by State or Province abbreviation. NE true
limit Number of responses to return. If used, must be a number between 1 and 10. Default is 10. 10 true
offset Number of responses to be skipped for generating the output. If used, must be a number between 0-100 minus limit. Default is 0. 0 true
Content-Type The content type. application/json false
