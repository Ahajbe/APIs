Pivotaltracker API: Get Stories

These methods are relative to the tag 'Get Stories'

get_story

Retrieves information about a single story.

[object Object]

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
STORY_ID The ID of the story
 false

get_all_stories

Retrieves all stories for a project, or those matching filter(s)

[object Object]

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
filter A filter for the returned stories to match. See the API documentation for more details.
 true
offset Controls the number of stories to skip from the beginning of the result.
 true
limit Controls the maximum number of stories returned.
 true
