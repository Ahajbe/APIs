Pivotaltracker API: Manage Projects

These methods are relative to the tag 'Manage Projects'

postAddProject

Adds a new project.

Authentication required

getAllMemberships

Retrieves all memberships for a project.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false

postAddMembership

Adds a new membership to a project.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false

getMembershipInfo

Retrieves information about a single membership.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
MEMBERSHIP_ID The ID of the membership
 false

deleteRemoveMembership

Removes a membership from a project

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
MEMBERSHIP_ID The ID of the membership
 false
