Pivotaltracker API: Activity Feed

These methods are relative to the tag 'Activity Feed'

get_activities

Retrieves the recent activity of all your projects.

[object Object]

Authentication required

Parameter name Description Default Optionnal
limit Limits the number of activity feed items. Note the default value is 10, and there is a upper cap of 100
 10 true
occurred_since_date Restricts the activity feed to only those items that occurred after a supplied date (example format: 2009/12/18 21:00:00 UTC)
 true
newer_than_version Restricts the activity feed to only those items that have a greater than supplied version
 true

get_project_activities

Retrieves the recent activity of a specific project.

[object Object]

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project to retrieve the activity for
 false
limit Limits the number of activity feed items. Note the default value is 10, and there is a upper cap of 100
 10 true
occurred_since_date Restricts the activity feed to only those items that occurred after a supplied date (example format: 2009/12/18 21:00:00 UTC)
 true
