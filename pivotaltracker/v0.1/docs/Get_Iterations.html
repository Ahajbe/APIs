Pivotaltracker API: Get Iterations

These methods are relative to the tag 'Get Iterations'

getIterations

Retrieves all iterations, with stories.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false

getIterationsDone

Retrieves iterations from the "done" group, with stories.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
offset Controls the number of iterations to skip from the beginning of the result.
 true
limit Controls the maximum number of iterations returned.
 true

getIterationsCurrent

Retrieves iterations from the "current" group, with stories.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false

getIterationsBacklog

Retrieves iterations from the "backlog" group, with stories.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
offset Controls the number of iterations to skip from the beginning of the result.
 true
limit Controls the maximum number of iterations returned.
 true

getIterationsCurrentBacklog

Retrieves iterations from the "current" and "backlog" groups, with stories.

Authentication required

Parameter name Description Default Optionnal
PROJECT_ID The ID of the project
 false
offset Controls the number of iterations to skip from the beginning of the result.
