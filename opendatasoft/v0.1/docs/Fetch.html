Open Data Soft API: Fetch

The Fetch API allowing to fetch the whole content of a dataset in a efficient way while still being able to use native filters.

fetchDatasets

Returns the collection of available datasets

Parameter name Description Default Optionnal
format the output format, one of json, jsonp and csv json true
callback the name of jsonp callback to be used in the response. none true
pretty_print when using json and jsonp, set the value of true to pretty_print the json output. true true

fetch

Fetches the whole content of a dataset (provided by datasetid)

Parameter name Description Default Optionnal
datasetid the id of the dataset (as returned by the datasets API calls) none false
format the output format, one of json, jsonp and csv json true
disps if set to geo, will be used to display geo coded datasets in an appropriate format {lat: XX, long: xx, data:{...}} none true
location a WGS84 point, indicating a geo position for geo filtering none true
callback the name of jsonp callback to be used in the response. none true
pretty_print when using json and jsonp, set the value of true to pretty_print the json output. true true
distance a geo distance expressed in meters true true
