Rdio API: Playlists

These methods are relative to the tag 'Playlists'

addToPlaylist

Adds a track to a playlist.

[object Object]

Authentication required

createPlaylist

Creates a new playlist in the current user's collection. The new playlist will be returned if the creation is successful, otherwise null will be returned.

[object Object]

Authentication required

deletePlaylist

Deletes a playlist.

[object Object]

Authentication required

getPlaylists

Gets the current user's playlists.

[object Object]

Authentication required

removeFromPlaylist

Removes items from a playlist by range (index and count). All track keys to remove must be in the tracks list too. This is to prevent accidental overwriting of playlist changes.

[object Object]

Authentication required

setPlaylistCollaborating

Starts or stops collaborating on this playlist.

[object Object]

Authentication required

setPlaylistCollaborationMode

Sets the playlist collaboration mode to allow no collaboration (0), collaboration with all Rdio users (1), or collaboration with Rdio users followed by the playlist owner (2).

[object Object]

Authentication required

setPlaylistFields

Sets the name and description for a playlist.

[object Object]

Authentication required

setPlaylistOrder

Saves the given order of tracks in a given playlist. The new order must have the same tracks as the previous order (this method may not be used to add/remove tracks).

[object Object]

Authentication required
