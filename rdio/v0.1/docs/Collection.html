Rdio API: Collection

These methods are relative to the tag 'Collection'

addToCollection

Adds tracks or playlists to the current user's collection.

Authentication required

getAlbumsForArtistInCollection

Gets the albums in the user's collection by a particular artist.

getAlbumsInCollection

Gets all of the albums in the user's collection.

getArtistsInCollection

Gets all of the artist in a user's collection.

getTracksForAlbumInCollection

Which tracks on the given album are in the user's collection.

getTracksForArtistInCollection

Which tracks from the given artist are in the user's collection.

getTracksInCollection

Gets all of the tracks in the user's collection.

removeFromCollection

Removes tracks or playlists from the current user's collection.

Authentication required

setAvailableOffline

Marks tracks or playlists for offline syncing.

Authentication required
