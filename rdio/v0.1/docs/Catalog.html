Rdio API: Catalog

These methods are relative to the tag 'Catalog'

getAlbumsByUPC

Finds and returns albums based on their Universal Product Code (UPC).

getAlbumsForArtist

Returns the albums by (or featuring) an artist.

[object Object]

getTracksByISRC

[object Object]

getTracksForArtist

search

[object Object]

searchSuggestions
