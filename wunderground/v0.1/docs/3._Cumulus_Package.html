Wunderground API: 3. Cumulus Package

These methods are relative to the tag '3. Cumulus Package'

forecast10Example

This example will return the 10 Day Forecast Summary for San Francisco, California

OneDayHourlyExample

This example will return a detailed hourly forecast for the 'next' 36 hours in San Francisco, California

satelliteExample

This example will return URL's for satellite (visual and IR) images for San Francisco, California

radarExample

This example will return dynamic radar image around Salt Lake City, UT with a transparent background. See Docs for all the options for creating these images.

radarExample2

This example will return dynamic radar image around Salt Lake City, UT with a Base Map background. See Docs for all the options for creating these images.

alertsExample

This example will return active severe alerts for San Francisco, California. This request only works in the USA, Canada and Europe.

TideExample1

This example will return 4 days of tides for San Francisco, California starting 'today'. This request only works in the USA.

TideExample2

This example will return 4 days of tides for San Francisco, California following March 3 2012. This request only works in the USA.

Tide_RAWExample

This example will return raw tide data for San Francisco, California on March 3 2016. This request only works in the USA.
