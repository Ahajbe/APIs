Wunderground API: 4. Anvil Package

These methods are relative to the tag '4. Anvil Package'

TenDayHourlyExample

This example will return a detailed hourly forecast for the next 10 days in San Francisco, California

yesterdayExample

This example will return yesterday's weather in San Francisco, California.

PlannerExample

This example will return a Historical roll up of weather data in San Francisco from Oct 5 to Oct 31. An example application: http://www.wunderground.com/travelplanner/index.asp

camsExample

This example will return URL's to webcam images in San Francisco, California. It will also return information about the webcam, including it's location in Lat,Long coordinates.

radarExample3

This example will return dynamic animated radar image around Salt Lake City, UT with a base map background. See Docs for all the options for creating these images.

radarExample4

This example will return dynamic animated radar image around Salt Lake City, UT with a transparent background. See Docs for all the options for creating these images.
