getCallList

Returns a list of phone calls made to and from the account identified by {AccountSid}, sorted by DateUpdated with most-recent calls first.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
To Only show calls to this phone number. undefined true
From Only show calls from this phone number. undefined true
Status Only show calls currently in this status. May be queued, ringing, in-progress, completed, failed, busy, or no-answer. undefined true
StartTime Only show calls that started on this date, given as YYYY-MM-DD. Also supports inequalities, such as StartTime(less than or equal to)YYYY-MM-DD for calls that started at or before midnight on a date, and StartTime(greater than or equal to)YYYY-MM-DD for calls that started at or after midnight on a date. undefined true
EndTime Only show calls that ended on this date, given as YYYY-MM-DD. Also supoprts inequalities, such as EndTime(less than or equal to)YYYY-MM-DD for calls that ended at or before midnight on a date, and EndTime(greater than or equal to)YYYY-MM-DD for calls that end at or after midnight on a date. undefined true
