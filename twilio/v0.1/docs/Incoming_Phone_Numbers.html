Twilio API: Incoming Phone Numbers

These methods are relative to the tag 'Incoming Phone Numbers'

getIncomingPhoneNumber

Fetches the details of a phone number purchased from Twilio

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id. undefined false
IncomingPhoneNumberSid A 34 character string that uniquely idetifies this resource. undefined false

updateIncomingPhoneNumberByPost

Update an incoming phone number obtained from Twilio via POST.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id. undefined false
IncomingPhoneNumberSid A 34 character string that uniquely idetifies this resource. undefined false
FriendlyName A human readable description of the new incoming phone number resource, with maximum length 64 characters. undefined true
ApiVersion Calls to this phone number will start a new TwiML session with this API version. Either 2010-04-01 or 2008-08-01. undefined true
VoiceUrl The URL that Twilio should request when somebody dials the phone number. undefined true
VoiceMethod The HTTP method that should be used to request the VoiceUrl. Either GET or POST. Defaults to POST. undefined true
VoiceFallbackUrl A URL that Twilio will request if an error occurs requesting or executing the TwiML defined by VoiceUrl. undefined true
VoiceFallbackMethod The HTTP method that should be used to request the VoiceFallbackUrl. Either GET or POST. Defaults to POST. undefined true
StatusCallback The URL that Twilio will request to pass status parameters (such as call ended) to your application. undefined true
StatusCallbackMethod The HTTP method Twilio will use to make requests to the StatusCallback URL. Either GET or POST. undefined true
SmsUrl The URL that Twilio should request when somebody sends an SMS to the new phone number. undefined true
SmsMethod The HTTP method that should be used to request the SmsUrl. Either GET or POST. Defaults to POST. undefined true
SmsFallbackUrl A URL that Twilio will request if an error occurs requesting or executing the TwiML defined by SmsUrl. undefined true
SmsFallbackMethod The HTTP method that should be used to request the SmsFallbackUrl. Either GET or POST. Defaults to POST. undefined true
VoiceCallerIdLookup Do a lookup of a caller's name from the CNAM database and post it to your app. Either true or false. Defaults to false. undefined true

getIncomingPhoneNumberList

Returns a list of incoming phone number resources, each representing a phone number given to your account

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id. undefined false

FriendlyName Only show the incoming phone number resources with friendly names that exactly match this name. undefined true
