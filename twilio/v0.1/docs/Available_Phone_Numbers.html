Twilio API: Available Phone Numbers

These methods are relative to the tag 'Available Phone Numbers'

getAvailableLocalPhoneNumberList

Returns a list of local AvailablePhoneNumber resource representations, each representing a phone number that is currently available for provisioning within your account.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
IsoCountryCode The ISO country code of this phone number undefined false
AreaCode Find phone numbers in the specified Area Code. Only available for North American numbers. undefined true
Contains A pattern to match phone numbers on. Valid characters are '*' and [0-9a-zA-Z]. The '*' character will match any single digit. undefined true
InRegion Limit results to a particular region (i.e. State/Province). Given a phone number, search within the same Region as that number. undefined true
InPostalCode Limit results to a particular postal code. Given a phone number, search within the same postal code as that number. undefined true
NearLatLong Given a latitude/longitude pair lat,long find geographically close numbers within Distance miles. undefined true
NearNumber Given a phone number, find a geographically close number within Distance miles. Distance defaults to 25 miles. undefined true
InLata Limit results to a specific Local access and transport area (LATA). Given a phone number, search within the same LATA as that number. undefined true
InRateCenter Limit results to a specific rate center, or given a phone number search within the same rate center as that number. Requires InLata to be set as well. undefined true
Distance Specifies the search radius for a Near- query in miles. If not specified this defaults to 25 miles. undefined true
