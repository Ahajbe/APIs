Twilio API: SMS Messages

These methods are relative to the tag 'SMS Messages'

sendSMS

Send a new outgoing message.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account. undefined false
From A Twilio number enabled for SMS. Only phone numbers purchased from Twilio work here; you cannot (for example) spoof SMS messages from your own cell phone number. undefined false
To The number to call. Format with a '+' and country code e.g., +16175551212 (E.164 format). Twilio will also accept unformatted US numbers e.g., (415) 555-1212, 415-555-1212. undefined false
Body The text of the message you want to send, limited to 160 characters. undefined false
StatusCallback A URL that Twilio will POST to when your message is processed. Twilio will POST the SmsSid as well as SmsStatus=sent or SmsStatus=failed. undefined true

getSMSList

Returns a list of SMS messages associated with your account identified by {AccountSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account undefined false
From Only show SMS messages from this phone number undefined true
To Only show SMS messages to this phone number. undefined true
DateSent Only show SMS messages sent on this date, given as YYYY-MM-DD. Example: DateSent=2009-07-06. You can also specify inequality, such as DateSent lt YYYY-MM-DD for SMS messages that were sent on or before midnight on a date, and DateSent gt YYYY-MM-DD for SMS messages sent on or after midnight on a date. undefined true

getSMS

Returns a single SMS message specified by the provided {SMSMessageSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account undefined false
SMSMessageSid Represents a 34 character string that uniquely identifies this SMSMessage undefined false
