Twilio API: Conferences

These methods are relative to the tag 'Conferences'

getConference

Returns a representation of the conference identified by {ConferenceSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
ConferenceSid A 34 character string that uniquely identifies this conference. undefined false

getParticipantListForConference

Returns the list of participants in the conference identified by {ConferenceSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
ConferenceSid A 34 character string that uniquely identifies this conference. undefined false
Muted Only show participants that are muted or unmuted. Either true or false. undefined true

getConferenceListForAccount

Returns a list of conferences within an account identified by {AccountSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
Status Only show conferences currently in with this status. May be init, in-progress, or completed. undefined true
FriendlyName List conferences who's FriendlyName is the exact match of this string. undefined true
DateCreated Only show conferences that started on this date, given as YYYY-MM-DD. You can also specify inequality, such as DateCreated(less than or equal to)YYYY-MM-DD for conferences that started at or before midnight on a date, and DateCreated(greater than or equal to)YYYY-MM-DD for conferences that started at or after midnight on a date. undefined true
DateUpdated Only show conferences that were last updated on this date, given as YYYY-MM-DD. You can also specify inequality, such as DateUpdated(less than or equal to)YYYY-MM-DD for conferences that were last updated at or before midnight on a date, and DateUpdated(greater than or equal to)YYYY-MM-DD for conferences that were updated at or after midnight on a date. undefined true
