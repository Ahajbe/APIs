Twilio API: Participants

These methods are relative to the tag 'Participants'

getParticipantForConference

Returns a representation of this participant.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
ConferenceSid A 34 character string that uniquely identifies this conference. undefined false
CallSid A 34 character string that uniquely identifies the call that is connected to this conference. undefined false

MuteParticipant

Mute a participant.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
ConferenceSid A 34 character string that uniquely identifies this conference. undefined false
CallSid A 34 character string that uniquely identifies the call that is connected to this conference. undefined false
Muted Specifying true will mute the participant, while false will un-mute. undefined true

deleteParticipant

Kick a participant from a conference.

Parameter name Description Default Optionnal
format json false

ConferenceSid A 34 character string that uniquely identifies this conference. undefined false
CallSid A 34 character string that uniquely identifies the call that is connected to this conference. undefined false

getParticipantListForConference2

Returns the list of participants in the conference identified by {ConferenceSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
ConferenceSid A 34 character string that uniquely identifies this conference. undefined false
Muted Only show participants that are muted or unmuted. Either true or false. undefined true
