Twilio API: Accounts

These methods are relative to the tag 'Accounts'

getAccount

Returns the account details for the {AccountSid}.

Parameter name Description Default Optionnal
format The response format. json false
AccountSid Represents the unique id of the Account undefined false

updateAccountByPost

Allows you to modify the 'FriendlyName' for the {AccountSid} via POST.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account undefined false
FriendlyName The human-readable description of an account. undefined true
Status Alter the status of this account: use closed to irreversibly close this account, suspended to temporarily suspend it, or active to reactivate it. undefined true

updateAccountByPut

Allows you to modify the 'FriendlyName' for the {AccountSid} via PUT.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account undefined false
FriendlyName The human-readable description of an account. undefined true
Status Alter the status of this account: use closed to irreversibly close this account, suspended to temporarily suspend it, or active to reactivate it. undefined true

getAccountList

Right now, your credentials are tied to one and only one account, so the list resource only contains your single account.

Parameter name Description Default Optionnal
format json false
FriendlyName Only return the Account resources with friendly names that exactly match this name. undefined true
Status Only return Account resources with the given status. Can be closed, suspended or active. undefined true

createSubAccount

Creates a new subaccount.

Parameter name Description Default Optionnal
format json false
FriendlyName A human readable description of the new subaccount, up to 64 characters. Defaults to "SubAccount Created at {YYYY-MM-DD HH:MM meridiam}". undefined true
