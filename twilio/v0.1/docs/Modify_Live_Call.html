Twilio API: Modify Live Call

These methods are relative to the tag 'Modify Live Call'

modifyLiveCall

Modifies a live call. Modification could be redirection or termination of the call.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
CallSid A 34 character string that uniquely idetifies this resource undefined false
Url A valid URL that returns TwiML. Twilio will immediately redirect the call to the new TwiML. undefined true
Method The HTTP method Twilio should use when requesting the above URL. Defaults to POST. undefined true
Status Either canceled or completed. Specifying canceled will attempt to hangup calls that are queued or ringing but not affect calls already in progress. Specifying completed will attempt to hang up a call even if it's already in progress. undefined true
