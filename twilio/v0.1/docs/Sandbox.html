Twilio API: Sandbox

These methods are relative to the tag 'Sandbox'

getSandbox

Returns the Sandbox resource associated with the account identified by {AccountSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false

updateSandboxByPost

Updates the Sandbox resource via POST.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
VoiceUrl [object Object] undefined true
VoiceMethod The HTTP method that should be used to request the above URL. Must be either GET or POST. Defaults to POST. undefined true
SmsUrl The URL that Twilio should request when somebody sends an SMS to the sandbox. undefined true
SmsMethod The HTTP method that should be used to request the SmsUrl. Must be either GET or POST. Defaults to POST. undefined true
