Twilio API: Recordings

These methods are relative to the tag 'Recordings'

getRecordingListForAccount

Returns a recording list associated with this {AccountSid}

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
CallSid Show only recordings made during the call given by this sid. undefined true
DateCreated Only show recordings created on the given date. Should be formatted as YYYY-MM-DD. You can also specify inequality, such as DateCreated(less than or equal to)YYYY-MM-DD for recordings generated at or before midnight on a date, and DateCreated(greater than or equal to)YYYY-MM-DD for recordings generated at or after midnight on a date. undefined true

getTranscriptionListForRecording

Each Recording instance resource has a Transcriptions subresource which represents the set of transcriptions generated from the recording (if any). Represents the set of transcriptions available for the recording identified by {RecordingSid}.

Parameter name Description Default Optionnal
format json false
AccountSid Represents the unique id of the Account responsible for this caller Id undefined false
RecordingSid The unique id of the Recording this Transcription was made of undefined false
