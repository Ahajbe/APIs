Echonest API: Sandbox

These methods are relative to the tag 'Sandbox'

sandbox.list

Lists detailed information about each asset in the sandbox.

[object Object]

Authentication required

Parameter name Description Default Optionnal
sandbox The name of the sandbox. false
callback The callback function for JSONP requests. Required if format is jsonp. true
results The number of results desired. 0 < results < 100 15 true
start The desired index of the first result returned. 15 true

sandbox.access

Gets access to the listed assets.

[object Object]

Authentication required

Parameter name Description Default Optionnal
sandbox The name of the sandbox. false
id The ID of the asset. false
oauth_nonce A random string of characters that differs from call to call and is never repeated in a 10 minute window. false
oauth_timestamp The number of seconds elapsed since midnight, 1 January 1970. Be sure this is within ten minutes of the real time. You can get the current timestamp that the Echo Nest API is using by via the oauth/timestamp method. false
oauth_signature_method Algorithm used to generate the signature. Supported methods are HMAC-SHA1. false
oauth_consumer_key The consumer key used to sign calls. false
oauth_signature The cryptographic signature for the method call. Calls are signed using your consumer key, your shared secret, the HTTP method and an encoded version of the base string of the URL. false
callback The callback function for JSONP requests. Required if format is jsonp. true
results The number of results desired. 0 < results < 100 15 true
start The desired index of the first result returned. 15 true
