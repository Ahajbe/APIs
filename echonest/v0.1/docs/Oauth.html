Echonest API: Oauth

These methods are relative to the tag 'Oauth'

oauth.timestamp

Returns the current time of The Echo Nest API server. This time can be used to synchronize the local time with The Echo Nest allowing you to set a proper timestamp used generate authenticated requests to the API.
