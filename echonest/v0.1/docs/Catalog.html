Echonest API: Catalog

These methods are relative to the tag 'Catalog'

catalog.create

Creates a catalog.

[object Object]

Authentication required

Parameter name Description Default Optionnal
name The name of the catalog. false
type The type of the catalog. false
callback The callback function for JSONP requests. Required if format is jsonp. true
Content-Type Specifies Content Type. multipart/form-data false

catalog.update

Updates (adds or deletes) items from a catalog.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id The ID of the catalog. false

callback The callback function for JSONP requests. Required if format is jsonp. true

catalog.read

Returns data stored in the catalog.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id The ID of the catalog. false
callback The callback function for JSONP requests. Required if format is jsonp. true
item_id The item id for the item in the catalog. true
bucket Indicates what data should be returned for each item.
For song items: audio_summary, artist_familiarity, artist_hotttnesss, artist_location, song_hotttnesss, tracks, id:Rosetta-space
For artist items: biographies, blogs, doc_counts, familiarity, hotttnesss, images, news, reviews, songs, terms, urls, video, years_active, id:Rosetta-space		 true
results The desired number of results returned. 30 true
start The desired index of the first result returned. 0 true

catalog.feed

Returns feeds based on the artists in a taste profile.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id The ID of the catalog. false
callback The callback function for JSONP requests. Required if format is jsonp. true
bucket Indicates what type of feed items should be returned for each artist in the catalog. news true
results The desired number of results returned. 25 true
start The desired index of the first result returned. 0 true
since Limit the items to those that have occurred since the given date. If supplied must be in date format YYYY-mm-dd true
high_relevance If true only items that are highly relevant for this artist will be returned. Currently only news items are filtered for high relevance. false true
