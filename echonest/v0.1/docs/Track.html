Echonest API: Track

These methods are relative to the tag 'Track'

track.analyze

Analyze a previously uploaded track with the latest version of the analyzer.

[object Object]

Authentication required

Parameter name Description Default Optionnal
callback The callback function for JSONP requests. Required if format is jsonp. true
wait If true wait for the analysis to be completed. true true
bucket Indicates which track data should be returned (only valid if wait is true). true
Content-Type Specifies Content Type. multipart/form-data false

track.profile

Get info about tracks given an id or md5.

[object Object]

Authentication required

Parameter name Description Default Optionnal
callback The callback function for JSONP requests. Required if format is jsonp. true
wait If true wait for the analysis to be completed. true true
bucket The type of track data that should be returned. true

track.upload.post

Upload a track to The Echo Nest's analyzer for analysis. Request with Content-Type "application/octet-stream", with the local file as the body of the request, and the parameters in the URL.

[object Object]

Authentication required

Parameter name Description Default Optionnal
url A url to an audio file. false
filetype Required if uploading a local file. The type of audio file to be analyzed. true
callback The callback function for JSONP requests. Required if format is jsonp. true
wait If true wait for the analysis to be completed. true true
bucket The type of track data that should be returned. true
Content-Type Specifies Content Type. application/octet-stream false

track.upload.post.query

Upload a track to The Echo Nest's analyzer for analysis. Request with Content-Type "multipart/form-data" where all parameters are in the post body and the track is in the "track" section of the post "files".

[object Object]

Authentication required

Parameter name Description Default Optionnal
url A url to an audio file. false
filetype Required if uploading a local file. The type of audio file to be analyzed. true
callback The callback function for JSONP requests. Required if format is jsonp. true
wait If true wait for the analysis to be completed. true true
bucket The type of track data that should be returned. true
Content-Type Specifies Content Type. multipart/form-data false
track The track data. false
