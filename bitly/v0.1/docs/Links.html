Bitly API: Links

These methods are relative to the tag 'Links'

shorten

Creates a new shortened link.

[object Object]

Authentication required

Parameter name Description Default Optionnal
longUrl Long URL to be shortened.
 false
domain bit.ly true

expand

Given a bitly URL or hash (or multiple), /v3/expand decodes it and returns back the target URL.

[object Object]

Authentication required

Parameter name Description Default Optionnal
shortUrl Short URL to be expanded.
 true
hash Refers to one or more bitly hashes.
 true

bitly_pro_domain

This is used to query whether a given short domain is assigned for bitly.Pro, and is consequently a valid shortUrl parameter for other api calls.

[object Object]

Authentication required

Parameter name Description Default Optionnal
domain A short domain.
 false

lookup

This is used to query for a bitly link based on a long URL.

[object Object]

Authentication required

Parameter name Description Default Optionnal
url One or more long URLs to lookup.
 false

info

This is used to return the page title for a given bitly link.

[object Object]

Authentication required

Parameter name Description Default Optionnal
shortUrl Refers to one or more short URLs.
 true
hash Refers to one or more bitly hashes.
 true
