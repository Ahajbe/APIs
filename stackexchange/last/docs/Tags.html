Stackexchange API: Tags

These methods are relative to the tag 'Tags'

getTags

Get the tags on the site.

Authentication required

Parameter name Description Default Optionnal
filter Required text in returned tags. string undefined true
fromdate Unix timestamp of the minimum creation date on a returned item. unix epoch date, range [0, 253,402,300,799] undefined true
max Maximum of the range to include in the response according to the current sort. integer or unix epoch date undefined true
min Minimum of the range to include in the response according to the current sort. integer or unix epoch date undefined true
order How the current sort should be ordered. one of desc (default), or asc desc true
page The pagination offset for the current collection. Affected by the specified pagesize. 32-bit signed integer undefined true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. 32-bit signed integer undefined true
sort How a collection should be sorted. one of activity (default), creation, or votes activity true
todate Unix timestamp of the maximum creation date on a returned item. Unix epoch date, range [0, 253,402,300,799] undefined true

getTagTopAnswerersByPeriod

Get the top answer posters in a specific tag, either in the last month or for all time. Returns the top 30 answerers.

Authentication required

Parameter name Description Default Optionnal
tag The tags in question. string undefined false
period One of all-time or month. string undefined false
