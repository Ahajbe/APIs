Stackexchange API: Revisions

These methods are relative to the tag 'Revisions'

getPostsRevisionsByIds

Get all revisions for posts identified by a set of ids.

Authentication required

Parameter name Description Default Optionnal
ids A single primary key identifier or a vectorized, semicolon-delimited list of identifiers. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon. string undefined false
fromdate Unix timestamp of the minimum creation date on a returned item. unix epoch date, range [0, 253,402,300,799] undefined true
todate Unix timestamp of the maximum creation date on a returned item. Unix epoch date, range [0, 253,402,300,799] undefined true

getPostSpecificRevision

Get a specific revision identified by a post id and a revision guid.

Authentication required

Parameter name Description Default Optionnal
id A single primary key identifier or a vectorized, semicolon-delimited list of identifiers. string undefined false
revisionguid specific revision id. guid, 8-4-4-4-12 format undefined false
fromdate Unix timestamp of the minimum creation date on a returned item. unix epoch date, range [0, 253,402,300,799] undefined true
todate Unix timestamp of the maximum creation date on a returned item. Unix epoch date, range [0, 253,402,300,799] undefined true
