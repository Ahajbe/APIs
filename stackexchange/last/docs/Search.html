Stackexchange API: Search

These methods are relative to the tag 'Search'

SearchQuestionsByCriteria

Search the site for questions meeting certain criteria.

Authentication required

Parameter name Description Default Optionnal
nottagged List of tags delimited by semi-colons of which at least one must be on a question. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon. string undefined true
max Maximum of the range to include in the response according to the current sort. integer or unix epoch date undefined true
min Minimum of the range to include in the response according to the current sort. integer or unix epoch date undefined true
order How the current sort should be ordered. one of desc (default), or asc desc true
page The pagination offset for the current collection. Affected by the specified pagesize. 32-bit signed integer undefined true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. 32-bit signed integer undefined true
sort How a collection should be sorted. one of activity (default), creation, or votes activity true

SearchSimilarQuestionsByTitle

Search the site based on similarity to a title.

Authentication required

Parameter name Description Default Optionnal
nottagged List of tags delimited by semi-colons of which at least one must be on a question. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon. string undefined true
tagged List of tags delimited by semi-colons of which at least one must be on a question. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon. string undefined true
answers When "true", the answers to a question will be returned. boolean undefined true
body When "true", a post's body will be included in the response. boolean undefined true
comments When "true", any comments on a post will be included in the response. boolean undefined true
title A title. string undefined true
