Stackexchange API: Privileges

These methods are relative to the tag 'Privileges'

privilegesList

Returns the earnable privileges on a site.

Privileges define abilities a user can earn (via reputation) on any Stack Exchange site.

While fairly stable, over time they do change. New ones are introduced with new features, and the reputation requirements change as a site matures.

Parameter name Description Default Optionnal
page The pagination offset for the current collection. Affected by the specified pagesize. true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. true
site stackoverflow false
