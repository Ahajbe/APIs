Stackexchange API: Access Tokens

These methods are relative to the tag 'Access Tokens'

accessTokensInvalidateGet

Immediately expires the access tokens passed. This method is meant to allow an application to discard any active access tokens it no longer needs.

Parameter name Description Default Optionnal
accessTokens {accessTokens} can contain up to 20 access tokens. These are obtained by authenticating a user using OAuth 2.0. false
page The pagination offset for the current collection. Affected by the specified pagesize. true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. true
