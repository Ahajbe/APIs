Stackexchange API: Revisions

These methods are relative to the tag 'Revisions'

revisionsGet

Returns edit revisions for the posts identified in id.

Parameter name Description Default Optionnal
id A single primary key identifier or a vectorized, semicolon-delimited list of identifiers. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon string undefined false
fromdate Unix timestamp of the minimum creation date on a returned item. undefined true
todate Unix timestamp of the maximum creation date on a returned item. undefined true

specificRevisionsGet

Gets a specific revision, as identified by post id and revisionguid.

Parameter name Description Default Optionnal
id A single primary key identifier or a vectorized, semicolon-delimited list of identifiers. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon string undefined false
revisionguid specific revision id, 8-4-4-4-12 format. undefined false
fromdate Unix timestamp of the minimum creation date on a returned item. undefined true
todate Unix timestamp of the maximum creation date on a returned item. undefined true
