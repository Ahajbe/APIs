Stackexchange API: Tags

These methods are relative to the tag 'Tags'

tagsList

Returns the tags found on a site.

Parameter name Description Default Optionnal
filter Required text in returned tags. undefined true
fromdate Unix timestamp of the minimum creation date on a returned item. undefined true
max Maximum of the range to include in the response according to the current sort. undefined true
min Minimum of the range to include in the response according to the current sort. undefined true
order How the current sort should be ordered. One of desc (default), or asc. undefined true
page The pagination offset for the current collection. Affected by the specified pagesize. undefined true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. undefined true
sort How a collection should be sorted. One of desc (default), or asc. undefined true
todate Unix timestamp of the maximum creation date on a returned item. undefined true

tagWikisGet

Returns the community editted tag wikis associated with a set of tags. Unlike privilege wikis, tag wikis carry an optional excerpt. Additionally, they are subject to much more frequent change.

Parameter name Description Default Optionnal
tags The tags for which to return synonyms. This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon. undefined false

topAnswerersInTagGet

Returns the top 30 answerers active in a single tag, of either all-time or the last 30 days. This is a view onto the data presented on the tag info page on the sites.

Parameter name Description Default Optionnal
period One of all-time or month. undefined false
tag The tag in question. undefined false

topAskersInTagGet

Returns the top 30 question askers active in a single tag, of either all-time or the last 30 days. This is a view onto the data presented on the tag info page on the sites.

Parameter name Description Default Optionnal
