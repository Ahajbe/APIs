Stackexchange API: Tags

These methods are relative to the tag 'Tags'

tagList

Returns the tags found on a site.

Parameter name Description Default Optionnal
filter Required text in returned tags. true
fromdate Unix timestamp of the minimum creation date on a returned item. true
max Maximum of the range to include in the response according to the current sort. true
min Minimum of the range to include in the response according to the current sort. true
order How the current sort should be ordered. One of desc (default), or asc. true
page The pagination offset for the current collection. Affected by the specified pagesize. true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. true
sort The sorts accepted by this method operate on the follow fields of the tag object:

popular – count
activity – the creation_date of the last question asked with the tag
name – name
popular is the default sort.
 true
todate Unix timestamp of the maximum creation date on a returned item. true
inname The inname parameter lets a consumer filter down to tags that contain a certain substring. For example, inname=own would return both "download" and "owner" amongst others.
true
site To specified on which site you want to search. stackoverflow (default) stackoverflow false

Parameter name Description Default Optionnal
tags For a question to be returned, it must have all the tags in {tags} and be considered "frequently asked". The exact algorithm for determining whether a question is considered a FAQ is subject to change at any time.

{tags} can contain up to 5 individual tags per request.		 false

tagRelatedGet

Returns the tags that are most related to those in {tags}.

Including multiple tags in {tags} is equivalent to asking for "tags related to tag #1 and tag #2" not "tags related to tag #1 or tag #2".

count on tag objects returned is the number of question with that tag that also share all those in {tags}.

{tags} can contain up to 4 individual tags per request.

This method returns a list of tags.

Parameter name Description Default Optionnal
tags The tags for which to return related.
This is a vectorized parameter, multiple ids can be sent a single request if delimited with a semi-colon.		 false
page The pagination offset for the current collection. Affected by the specified pagesize. true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. true
site To specified on which site you want to search. stackoverflow (default) stackoverflow false
