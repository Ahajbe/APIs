GETnotificationsList

Returns a user's notifications.

This method requires an access_token, with a scope containing "read_inbox".

This method returns a list of notifications.

Default
Description
Parameter
string

The pagination offset for the current collection. Affected by the specified pagesize.

page

Optional

string

The number of collection results to display during pagination. Should be between 0 and 100 inclusive.

pagesize
