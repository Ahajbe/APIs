Stackexchange API: Notifications

These methods are relative to the tag 'Notifications'

notificationsList

Returns a user's notifications.

This method requires an access_token, with a scope containing "read_inbox".

This method returns a list of notifications.

Authentication required

Parameter name Description Default Optionnal
page The pagination offset for the current collection. Affected by the specified pagesize. true
pagesize The number of collection results to display during pagination. Should be between 0 and 100 inclusive. true
