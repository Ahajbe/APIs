GETsuggestedEdit

Returns all the suggested edits in the systems.

This method returns a list of suggested-edits.

Test console
Description
Parameter
string

Unix timestamp of the minimum creation date on a returned item.

fromdate

Optional

string

Unix timestamp of the maximum creation date on a returned item.

todate

Optional

string

Maximum of the range to include in the response according to the current sort.

max

Optional

string

Minimum of the range to include in the response according to the current sort.

min

Optional

string

How the current sort should be ordered. One of desc (default), or asc.

order

Optional

string

The pagination offset for the current collection. Affected by the specified pagesize.

page

Optional

string

The number of collection results to display during pagination. Should be between 0 and 100 inclusive.

pagesize

Optional

string

The sorts accepted by this method operate on the follow fields of the suggested_edit object:

creation – creation_date
approval – approval_date
Does not return unapproved suggested_edits
rejection – rejection_date
Does not return unrejected suggested_edits
creation is the default sort.

sort

Optional

string

To specified on which site you want to search. stackoverflow (default)

site
