Contextio API: OAuth Providers

These methods are relative to the tag 'OAuth Providers'

listOauthProviders

List configured OAuth providers.

[object Object]

Authentication required

addOauthProviders

Adds a new OAuth provider.

[object Object]

Authentication required

Parameter name Description Default Optionnal
type Identification of the OAuth provider. This must be either GMAIL and GOOGLEAPPSMARKETPLACE. false
provider_consumer_key The OAuth consumer key. false
provider_consumer_secret The OAuth consumer secret. false

getOauthProvider

Gets information about a given OAuth provider.

[object Object]

Authentication required

Parameter name Description Default Optionnal
key The consumer key for this external OAuth provider. false

removeOauthProvider

Removes a given OAuth provider.

[object Object]

Authentication required

Parameter name Description Default Optionnal
key The consumer key for this external OAuth provider. false
