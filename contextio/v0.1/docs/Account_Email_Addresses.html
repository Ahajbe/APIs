Contextio API: Account Email Addresses

These methods are relative to the tag 'Account Email Addresses'

listAccountEmailAddresses

Lists email addresses used by an account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false

addAccountEmailAddress

Adds a new email address as an alias for an account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
email_address An email address. false

removeAccountEmailAddress

Removes an email address form the aliases of an account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
email One of the email addresses associated to the account. false

setAccountEmailAddressAsPrimary

Makes this email address the primary one for the account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
email One of the email addresses associated to the account. false
primary Set to 1 to make this email address the primary one for the account. 1 false
