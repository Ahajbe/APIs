Contextio API: Accounts

These methods are relative to the tag 'Accounts'

listAccounts

Lists accounts.

[object Object]

Authentication required

Parameter name Description Default Optionnal
token The unique random token used for the graphical account creation process. false
email Only return account associated to this email address. true
status Only return accounts with sources whose status is of a specific value. If an account has many sources, only those matching the given value will be included in the response. Possible statuses are: INVALID_CREDENTIALS, CONNECTION_IMPOSSIBLE, NO_ACCESS_TO_ALL_MAIL, OK, TEMP_DISABLED and DISABLED. true
status_ok Set to 0 to get all accounts with sources that are not working correctly. Set to 1 for the opposite. As for the status filter above, only sources matching the specific value are included in the response. true
limit The maximum number of results to return. true
offset Start the list at this offset (zero-based). true

addAccount

Adds a new account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
token The unique random token used for the graphical account creation process. false
email The primary email address of the account holder. false
first_name First name of the account holder. true
last_name Last name of the account holder. true
