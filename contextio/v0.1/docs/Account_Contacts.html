Contextio API: Account Contacts

These methods are relative to the tag 'Account Contacts'

listAccountContacts

Lists contacts in an account.

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
search String identifying the name or the email address of the contact(s) you are looking for. true
active_before Only include contacts included in at least one email dated before a given time. This parameter should be a standard unix timestamp. true
active_after Only include contacts included in at least one email dated after a given time. This parameter should be a standard unix timestamp. true
limit The maximum number of results to return. true
offset Start the list at this offset (zero-based). true

getAccountContact

Gets information about a given contact.

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
email Email address of a contact. false

listAccountContactFiles

Lists files exchanged with a contact. Returns the latest attachments exchanged with one or more email addresses. By "exchanged with Mr. X" we mean any file attached to an email received from Mr. X, sent to Mr. X or sent by anyone to both Mr. X and the mailbox owner.

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
email Email address of a contact. false
limit The maximum number of results to return. true
offset Start the list at this offset (zero-based). true

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
