Contextio API: Account Connect Tokens

These methods are relative to the tag 'Account Connect Tokens'

listAccountConnectTokens

Lists connect tokens created for an account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false

obtainNewAccountConnectToken

Obtains a new connect_token for a specific account.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
callback_url When the user's mailbox is connected to your API key, the browser will call this url (GET). This call will have a parameter called contextio_token indicating the connect_token related to this callback. You can then do a get on this connect_token to obtain details about the account and source created through that token and save that account id in your own user data. false
service_level Sets the service level of the account's source to be created. Possible values are PRO and BASIC. true
email The email address of the account to be added. If specified, the first step of the connect UI where users are prompted for their email address, first name and last name is skipped. true
first_name First name of the account holder. true
last_name Last name of the account holder. true

getAccountConnectToken

Gets information about a given connect token.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
token The unique random token used to add a second source to an existing account. false

removeAccountConnectToken

Remove a given connect token.

[object Object]

Authentication required

Parameter name Description Default Optionnal
id Unique id of an account accessible through your API key. false
token The unique random token used to add a second source to an existing account. false
