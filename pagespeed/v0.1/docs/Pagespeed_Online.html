Pagespeed API: Pagespeed Online

These methods are relative to the tag 'Pagespeed Online'

run

Runs Page Speed analysis on the page at the specified URL, and returns a Page Speed score, a list of suggestions to make that page faster, and other information.

Authentication required

Parameter name Description Default Optionnal
url The URL to fetch and analyze. false
locale The locale used to localize formatted results. en_US true
rule A Page Speed rule to run; if none are given, all rules are run. true
strategy The analysis strategy to use. desktop true
callback Callback function. true
prettyPrint Returns response with identations and line breaks. true true
userIp IP address of the site where the request originates. true
