Mailchimp API: Email Verification

These methods are relative to the tag 'Email Verification'

sts.delete.verified.email.address

Deletes a verified email address. This action takes immediate effect, so use it with care.

[object Object]

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
email The campaign id to get content. false

sts.list.verified.email.addresses

Returns a list containing all of the email addresses that have been verified.

sts.verify.email.address

Verifies an email address. This action causes a confirmation email message to be sent to the specified address.
