Mailchimp API: Stats

These methods are relative to the tag 'Stats'

sts.get.send.quota

Returns the user's current activity limits.

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false

sts.get.send.statistics

Returns the user's sending statistics. The result is a list of data points, representing the last two weeks of sending activity. Each data point in the list contains statistics for a 15-minute interval.

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
