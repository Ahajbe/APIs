Mailchimp API: Campaign Report Data

These methods are relative to the tag 'Campaign Report Data'

campaign.report.data.click.detail.aim

Return the list of email addresses that clicked on a given url, and how many times they clicked.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. campaignClickDetailAIM false
cid The campaign id to get stats false
url The URL of the link that was clicked on. false
start For large data sets, the page number to start at - defaults to 1st page of data (page 0). 0 true
limit For large data sets, the number of results to return - defaults to 1000, upper limit set at 15000. 1000 true

campaign.report.data.email.stats.aim

Given a campaign and email address, return the entire click and open history with timestamps, ordered by time.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. campaignEmailStatsAIM false
cid The campaign id to get stats. false
email_address An array of up to 50 email addresses to check OR the email "id" returned from listMemberInfo, Webhooks, and Campaigns. For backwards compatibility, if a string is passed, it will be treated as an array with a single element (will not work with XML-RPC). false
