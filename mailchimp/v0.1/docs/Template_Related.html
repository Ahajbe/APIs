Mailchimp API: Template Related

These methods are relative to the tag 'Template Related'

template.add

Create a new user template, NOT campaign content. These templates can then be applied while creating campaigns.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. templateAdd false
name The name for the template - names must be unique and a max of 50 bytes. false
html A string specifying the entire template to be created. This is NOT campaign content. false

template.list

Retrieve various templates available in the system, allowing some thing similar to our template gallery to be created.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. templates false
