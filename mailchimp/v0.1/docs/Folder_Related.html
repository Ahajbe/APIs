Mailchimp API: Folder Related

These methods are relative to the tag 'Folder Related'

folder.related.add

Add a new folder to file campaigns or autoresponders.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. folderAdd false
name A unique name for a folder (max 100 bytes). false
type The type of folder to create. campaign true

folder.related.del

Delete a campaign or autoresponder folder. Note that this will simply make campaigns in the folder appear unfiled, they are not removed.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. folderDel false
fid The folder id to update. false
type The type of folder to create. campaign true

folder.related.update

Update the name of a folder for campaigns or autoresponders.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. folderUpdate false
fid The folder id to update. false
name A new, unique name for a folder (max 100 bytes). false
