Mailchimp API: Export

These methods are relative to the tag 'Export'

list.export

Exports/dumps members of a list and all of their associated details. This is a very similar to exporting via the web interface.

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
id The list id to get members. false
status The status to get members. subscribed true
segment Pull only a certain Segment of your list. true
since Only return member whose data has changed since a GMT timestamp ��� in YYYY-MM-DD HH:mm:ss format. true

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
id The campaign id to retrieve stats. false
include_empty If set to "true" a record for every email address sent to will be returned even if there is no activity data. false true
