Mailchimp API: Campaign Related

These methods are relative to the tag 'Campaign Related'

campaign.related.get.content

Get the content (both html and text) for a campaign either as it would appear in the campaign archive or as the raw, original content.

[object Object]

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
method Is the method name from our API you are calling. campaignContent false
cid The campaign id to get content for (can be gathered using campaigns()). false
for_archive Controls whether we return the Archive version (true) or the Raw version (false), defaults to true. true true
