Mailchimp API: MC Stats

These methods are relative to the tag 'MC Stats'

sts.get.bounces

Returns all full bounce messages from the past 48 hours.

[object Object]

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
since If a time is provided, retrieve bounce messages since that time. Y-m-d H:i:s format. true

sts.get.send.stats

Returns all stats in one hour intervals.

[object Object]

Authentication required

Parameter name Description Default Optionnal
dc The portion after the dash in your API Key. e.g. "us1", "us2", "uk1", etc. false
apikey A valid API Key for your user account. false
tag_id Whether to limit to a single tag. true
since Limit data returned since this data/hour combo. This is a string comparison, not a date comparison, and we are not attempting to strictly validate this, so if you pass in something silly, expect silly results. It expects the same format as the returned data: "YYYY-MM-DD HH". true
