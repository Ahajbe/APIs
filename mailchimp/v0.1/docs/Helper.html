Mailchimp API: Helper

These methods are relative to the tag 'Helper'

helper.campaigns.for.mail

Retrieve all Campaigns Ids a member was sent.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. campaignsForEmail false
email_address The email address to unsubscribe OR the email "id" returned from listMemberInfo, Webhooks, and Campaigns. false
options Extra options to modify the returned data. See options: "http://apidocs.mailchimp.com/api/1.3/campaignsforemail.func.php" true

helper.generate.text

Have HTML content auto-converted to a text-only format. You can send: plain HTML, an array of Template content, an existing Campaign Id, or an existing Template Id. Note that this will not save anything to or update any of your lists, campaigns, or templates.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. generateText false
type The type of content to parse. false
content The content to use. For "html" expects a single string value, "template" expects an array like you send to campaignCreate, "url" expects a valid and public URL to pull from, "cid" expects a valid Campaign Id, and "tid" expects a valid Template Id on your account. false
