Mailchimp API: Campaign Stats

These methods are relative to the tag 'Campaign Stats'

campaign.stats.abuse.reports

Get all email addresses that complained about a given campaign.

Authentication required

Parameter name Description Default Optionnal
dc the portion after the dash in your api key. e.g. "us1", "us2", "uk1", etc. false
apikey a valid api key for your user account. false
method is the method name from our api you are calling. campaignAbuseReports false
cid The campaign id to pull abuse reports. false
start For large data sets, the page number to start at - defaults to 1st page of data (page 0). 0 true
limit For large data sets, the number of results to return - defaults to 500, upper limit set at 1000. 500 true
since Pull only messages since this time - use YYYY-MM-DD HH:II:SS format in GMT. 500 true
