Pinboard API: Tags

These methods are relative to the tag 'Tags'

getTags

Returns a full list of the user's tags along with the number of times they were used.

Authentication required

deleteTags

Delete an existing tag.

Authentication required

Parameter name Description Default Optionnal
tag The tag to delete. false

renameTag

Rename an tag, or fold it in to an existing tag.

Authentication required

Parameter name Description Default Optionnal
old The tag to rename. Note: match is not case sensitive. false
new The new name. If empty, nothing will happen. false
