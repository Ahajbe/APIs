Pinboard API: Bookmarks

These methods are relative to the tag 'Bookmarks'

postAdd

Add a new bookmark.

Authentication required

Parameter name Description Default Optionnal
url The URL of the item. false
description Title of the item. This field is unfortunately named 'description' for backwards compatibility with the delicious API. false
extended Description of the item. Called 'extended' for backwards compatibility with delicious API. true
tags List of up to 100 tags. true
dt Creation time for this bookmark. Defaults to current time. Datestamps more than 10 minutes ahead of server time will be reset to current server time. true
replace Replace any existing bookmark with this URL. Default is yes. If set to no, will throw an error if bookmark exists. yes true
shared Make bookmark public. Default is "yes" unless user has enabled the "save all bookmarks as private" user setting, in which case default is "no". yes true
toread Marks the bookmark as unread. Default is "no". no true

postsDelete

Delete a bookmark.

Authentication required

Parameter name Description Default Optionnal
url The URL of the item. false

postGet

Returns one or more posts on a single day matching the arguments. If no date or url is given, date of most recent bookmark will be used.

Authentication required

Parameter name Description Default Optionnal
tag Filter by up to three tags. true
dt Return results bookmarked on this day. true
url Return bookmark for this URL. true
meta Include a change detection signature in a meta attribute (yes/no). yes true

postDates

Returns a list of dates with the number of posts at each date.

Authentication required

Parameter name Description Default Optionnal
tag Filter by up to three tags. true

postRecent

Returns a list of the user's most recent posts, filtered by tag.

Authentication required

Parameter name Description Default Optionnal
tag Filter by up to three tags. true
count Number of results to return. Default is 15, max is 100. 15 true

postAll

Returns all bookmarks in the user's account.

Authentication required

Parameter name Description Default Optionnal
tag Filter by up to three tags. true
start Offset value (default is 0). 0 true
results Number of results to return. Default is all. true
fromdt Return only bookmarks created after this time. true
todt Return only bookmarks created before this time. true
meta Include a change detection signature for each bookmark. true

postSuggest

Returns a list of popular tags and recommended tags for a given URL. Popular tags are tags used site-wide for the url; recommended tags are drawn from the user's own tags.

Authentication required

Parameter name Description Default Optionnal
url The URL of the item. false
